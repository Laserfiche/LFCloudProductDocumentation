New Features in Laserfiche Cloud 2019.3

The Laserfiche Cloud 2019.3 update, released on July 11, 2019, introduces the Laserfiche home screen.

Watch an overview of the new features here:

Laserfiche Home

The Laserfiche home screenLaserfiche home screen provides a centralized dashboard where you can quickly access active and high priority tasks, starred and recent documents, and forms. Additional tabs are available for extended access, depending on your account settings.

The documents tab provides five configurable widgets for quick access to specific folder contents or starred documents, and a Document Search widget for repository searches, including saved search functionality.

The tasks tab extends user access to all available tasks, provides task search capability, review of completed tasks, and access to Team tasks.

The reports tab enables access to reporting information for business process and repository information.

The submit forms tab allows you to select a form to fill out, from either thumbnail or list views.

Process Automation

Business Processes

Laserfiche Forms features are now available as part of the Process Automation suite as Business ProcessesBusiness Processes. This includes the process designer, where you can create form-based review and approval processes. Additionally, the new Tasks, Submit Forms, and Reports tabs on the Laserfiche home screen provide easy user access to the processes.

Surveys

SurveysSurveys is a new feature that enables you to design custom surveys, polls, or registration forms to automatically collect information and view results without creating processes or designing reports.

Integrations

The IntegrationsIntegrations component in Process Automation combines CRM connections, external data sources, and remote agents to make them easier to manage and find.

The Salesforce integration has been enhanced, and allows for access to custom objects created in Salesforce. You can now create a CRM Connection to Salesforce that recognizes the custom fields of the object, and then use those in a workflow with the Perform CRM ActionPerform CRM Action activity to create, delete, update, and modify accounts in Salesforce.

Capture Profiles

Auto-classification with Capture Profiles automatically selects the profile that best matches the document and processes it accordingly. This creates an automated and more streamlined document capture and classification solution without having to manually add or maintain classification conditions.

Laserfiche App

The Laserfiche app now allows signing in to Laserfiche Cloud using Face, Touch, and Fingerprint ID on Android and iOS.

Users no longer need to download an entire PDF before viewing it when using Android.

Laserfiche Web Client

Administrators can now specify an OCR language for the repository, which will increase OCR accuracy for documents in that language. In addition, Audit Trail reporting now has a more modern and easy-to-use interface.
