New Features in Laserfiche Cloud 2019.4

The Laserfiche Cloud 2019.4 update, released on October 10, 2019, includes improved collaboration features.

Watch an overview of the new features here:

Documents

Co-authoring with Microsoft Office 365

Co-author Microsoft Word, Excel, or PowerPoint documents with multiple people using your Office 365 Business account. Changes can be made directly in your web browser, and will be automatically integrated and saved in the repository.

Laserfiche Vault

This release introduces Laserfiche Vault, an add-on to Cloud that enables strict compliance mode for records management.

OCR Language Support

Laserfiche Cloud automatically performs OCR on all documents scanned or imported into the repository. You can now set an OCR language for documents on a repository-wide basis or configure it directly on an individual document at the time of import. The OCR language set at the document level will take precedence over repository-wide settings, allowing you to easily support a repository with documents in multiple languages.

Process Automation

Public Facing Forms in Process Automation

Collect information from public users with web forms to kick off internal processes in Process Automation. Public forms can be shared with users whether they are registered or not, allowing organizations to collect and act on responses from anyone. With public portal, you can turn this option on when viewing the process details for a business process.

Integrations

Microsoft Dynamics has been added to the list of built-in CRM platform integrations, and provides access to standard objects in the Dynamics 365 Sales Hub through the Workflow "Perform CRM Action" activity. Data query rules now include options to Insert and Update data in data sources. Define Web Service Connections and Web Request Rules, for use with the Run Web Request Rule activity, to build integrations with external systems.
