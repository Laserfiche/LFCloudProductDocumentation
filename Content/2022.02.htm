February 2022 Updates

The February 2022 update, released on February 3, 2022, includes the following new features.

Watch an overview of the new features here:

Direct Share Enhancements

When sharing documents via direct share, the sender’s name is now included in the email subject line by default, letting recipients more easily recognize incoming files.

In-App User Feedback

You can share feedback and suggestions with the Laserfiche product team directly from the Laserfiche Cloud interface.

Expanded Form Layout Designer Capabilities

Built-in options added to the form layout designer allow for pagination and the ability to view uploaded image files directly on the form.

Watermarks for Public Portal Documents

A new feature in Public Portal allows you to apply a default watermark to repository content. Use the watermark to indicate, for example, where and when a document was generated, thus further enabling a degree of accountability and auditing.

New Solution Templates

Accelerate solution deployment with new and updated prebuilt templates including COVID-19 Test Result Submission, COVID-19 Vaccination Exemption Request, New Vendor Request, Accounts Payable - Purchase Order Requisition, and K-12 New Hire Onboarding.
