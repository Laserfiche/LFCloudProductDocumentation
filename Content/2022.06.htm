June 2022 Updates

The June 2022 update, released on June 9, 2022, includes the following new features.

Watch an overview of the new features here:

Transcription

Laserfiche now allows you to search within the audio content of media files. When an audio or video file is imported into Laserfiche, it will be automatically transcribed. The resulting transcription can then be searched like any other document with associated text. Context hits allow users to jump directly to the relevant part of the media file, and they can also view the text side-by-side with the file. Video and audio files imported on or after June 9, 2022 will automatically be transcribed. Media files that were imported before that date need to be manually transcribed to become searchable or to be viewed alongside the video.

Laserfiche API Client Libraries

Access the Laserfiche API using new client libraries. TypeScript/JavaScript and .NET client library packages for OAuth authentication and working with a Laserfiche Cloud repository are available through the npm registry and the NuGet gallery. The libraries are open source and the implementations and sample projects are available on GitHub.

Submit Your Own Solution

New solution templates include processes for collecting information related to the federal assistance program as well as an invoice review and payment solution that makes use of smart invoice capture.

You can now contribute to the Solution Marketplace by submitting your own solutions. From the marketplace, click the Submit a Solution button. You can upload your solution directly from this form and a Laserfiche Solution Marketplace team member will reach out to guide you through the review process and help launch your solution.

Expansion of Integrations Activities

All application connection integrations now support generic API querying, allowing for a much broader range of integration possibilities when making calls to Salesforce, Redtail, Dynamics, and Ellucian Ethos during Business Processes.

Colleague and Banner Finance Integrations

In the specific case of Ellucian Ethos, the application connection updates add support for integrations with Colleague and Banner Finance in addition to Banner.

Bulk Terminate Workflows

You now have the ability to terminate multiple workflow instances when performing account maintenance and clean up

Azure Active Directory SCIM Support

Minimize manual administration work by provisioning and deprovisioning users in Laserfiche Cloud automatically when administrators make changes in Azure Active Directory.

Generate User Listing Report

Download a full listing of users by using the "Generate user report" option to save a .CSV report containing all users along with additional information for each user, for example, email, assigned license, last login time, and more.
