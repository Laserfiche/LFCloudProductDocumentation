August 2022 Updates

The August 2022 update, released on August 4, 2022, includes the following new features.

Watch an overview of the new features here:

Application Connections for Team Projects

Connect third-party applications to Teams, so that you can use CRM connections in a secured project environment. View a project's application connections by visiting a team's project workspace and selecting Integrations.

Business Process Updates

There are a variety of updates to business processes.

  • Manage and monitor public, private, and task drafts to help users recover saved draft form submissions and delete drafts from the Cloud.
  • Use dynamic form selection to assign a form for a task based on a variable within a User Task. This lets a previous selection in the process determine what form should be presented to the user assigned to the task.
  • Send email notifications when a task is assigned to a user, reassigned to a user, or both.
  • New User Task variables are available to use throughout the process once the task has been completed. For example, you can customize email task links with variables.
  • New Save to repository variables are available to access the saved data.
  • Edit the Reply To, CC, and BCC addresses for all email notifications like email service tasks and user task notifications within a business process.

Solution Marketplace Update

Accelerate solution deployment with a new template for student registration and enrollment.
