September 2022 Updates

The September 2022 update, released on September 1, 2022, includes the following new features.

Watch an overview of the new features here:

Business Processes

A variety of updates to business processes improve security, form signatures, and emails.

  • When using the signature field in a form, a new option lets users save their signature so they can apply the same signature to future forms. You can turn this option on when configuring the Signature field in the Form designer.
  • You can now require email address confirmation when saving a draft of a form. This helps ensure that email addresses are correct.
  • Email size limits have been increased.

Process Automation Security Updates

  • Process administrators can now use project security to secure the access and edit rights for web requests.
  • A new security feature lets you hide step history information from users who do not need to see the actions that others may take. Steps can be hidden at the start event and activity level, or at the overall process level.

Workflow Updates

  • An updated workflow import process now allows you to map each referenced capture profile to a capture profile in the new environment. The import process also has an improved flow for configuring these mappings.
  • Email size limits have been increased.

Solution Marketplace

  • Community-submitted solution templates that were originally only available for self-hosted Laserfiche implementations are now available in Laserfiche Cloud. These templates let you:
    • Create a Laserfiche resource navigation page
    • Create a process that allows users to request changes to existing processes
    • Submit requests for ordering payment and/or travel cards
    • Request access for new employees for email, a phone, the enterprise ERP, and other systems
    • Streamline the hiring process with a template for new student hires
  • The Solution Marketplace has been updated with icons to show the organization that submitted the solution. Laserfiche-created solutions have an icon showing the solution’s category.
  • The complexity ranking was removed from the Solution Marketplace and replaced with a list of the components included in each solution.
