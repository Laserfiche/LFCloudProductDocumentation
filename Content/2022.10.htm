October 2022 Updates

The October 2022 update, released on October 6, 2022, includes the following new features.

Watch an overview of the new features here:

Layout Designer Enhancements

Conditionally apply validation to fields depending on values set in other fields. For example, you can configure validation to guide users to choose an end date that is after the start date they set in a previous field.

You can add labels to form table rows to help guide users through filling out tables.

On submitted forms, uploaded images and PDFs are easier to preview with a new eye icon next to the image. From the preview window you can zoom in on the image and review it without having to download it.

The Form Designer now has all the functionality of the Classic Form Designer and will be the default when creating new forms. The new designer includes all recent designer updates such as custom JavaScript support, dynamic validation, enhanced field rules, etc

Process Automation Updates

The Save to Repository service task provides new variables for saved forms. These include variables for the Folder Attachment List, Folder ID, Folder Link, Folder Name, and Folder Path.

Solution Marketplace

New community-submitted solution templates are now available that let you:

  • Automate workflows for During Product Inspection.
  • Enable K-12 schools or school districts to collect student vaccination status from students or parents.
  • Assist medical examiners to gather pertinent info in the field.
  • Submit requests for new student hires.
  • Submit requests for student job title and/or payrate changes.
  • Report property damage that occurs in the community.

If you are interested in sharing your solutions with the wider Laserfiche community, please visit the Laserfiche Solution Marketplace and follow the Submit a Solution link at the top of the page.
