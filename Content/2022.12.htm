December 2022 Updates

The December 2022 update, released on December 1, 2022, includes the following new features.

Watch an overview of the new features here:

Handwriting Recognition and Improved OCR Accuracy

Capture Profiles now provides support for capturing handwritten text, and users will also see significantly improved accuracy for printed characters. This streamlines configuration and maintenance for existing capture processes and opens up new opportunities by supporting a wider variety of document types.

Script Rules in Projects

Script Rules can now be created and secured within a project. This makes script rules available to processes created within that project.

Workflow's Search Activity Expanded

You can now search for entries based on their name through the Search Repository activity.

Search and Pagination in Forms Tables and Collections

If you have a form that includes tables or collections with lots of rows or sets, you can now paginate the tables and collections based on their sizes. For example, you can stipulate that a new page starts for a table if it has more than four rows. Additionally, you can add a search box to tables or collections so users can quickly find specific values in large tables or collections.

Upload Images in Custom HTML Form Fields

You can now upload images into custom HTML form fields, letting you further customize your form to fit your message and brand.

New Solution Templates

New community-submitted solution templates are now available that let you:

  • Manage new contract submissions that goes through a review and approval process.
  • Export metadata from a repository based on a reporting services request.
  • Allow staff to provide feedback directly to the company president or executive anonymously.
  • Create capture profiles for tax forms that can be used to automate data capture for human resources staff.
