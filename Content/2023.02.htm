February 2023 Updates

The February 2023 update, released on February 2, 2023, includes the following new features.

Watch an overview of the new features here:

Elevated Developer Role

Users with the Team Developer role who have Process Automation access can now develop within a team’s projects. Previously users would need an account-wide Process Automation Developer role in order to do so. This update lets users develop within a project without being able to see Global items.

Global Resource Sharing Control Updates

A project listing page is now available under the Teams section in Process Automation which allows Process Administrators to manage what Global resources projects may have access to as well as see all projects that are within their Laserfiche system.

Layout Designer Updates

More methods have been added to the LFForm object to allow changing field settings, form settings, field rules, and adding rows and sets to tables and collections. Additionally, two events have been added for lookup trigger and lookup done.

When there are fields on the form with duplicate names, sometimes it is not clear which field is associated with a field ID or variable name. Now when you are in the rules tab, the info tag for each field appears.

Account ID Added to Laserfiche URLs

Laserfiche URLs created from tasks in business processes or from the Copy URL option in the repository now include the account ID. That means when a Laserfiche user clicks these URLs, the account number auto-populates on the Laserfiche sign-in page for a smoother user experience.

New Solution Templates

New solution templates include a sample integration with Google Cloud’s Sentiment Analysis API, a network access request, and an early identification of mental illness process.
