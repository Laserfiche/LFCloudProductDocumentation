March 2023 Updates

The March 2023 update, released on March 2, 2023, includes the following new features.

Watch an overview of the new features here:

Laserfiche Connector for Boomi

Through our strategic partnership with the Integration Platform, Boomi, you can now easily connect Laserfiche's content management system and process automation to over 100 applications that your company runs on. Our Connector is built on top of the Laserfiche API, so you can connect both Laserfiche self-hosted and cloud systems, making it easy to view, save, update, and download Laserfiche documents and metadata all from another application. In addition, Laserfiche business processes can connect and share documents and metadata into the Boomi platform, extending the power of Laserfiche process automation across the entire organization. Explore the Laserfiche Solution Marketplace today to learn more about the platform connectors available.

Customize Forms Email Display Name

You can create custom sender display names for email messages sent as part of a business process. By default, business process email notifications are shown as being sent by Laserfiche Notification. Process designers can select additional display names when configuring the email services or user task.

OAuth 2.0 Scope Support

OAuth 2.0 scope support has been added for additional security when using the Laserfiche APIs. Scopes allow administrators to indicate what rights can be performed by someone using this connection. See the Laserfiche Developer Center, https://developer.laserfiche.com, for information on how to use scopes.

Laserfiche Developer Console Update

In the Laserfiche Developer Console, an option has been added for service applications to create a long-lasting key for authentication. This can be used by applications that are unable to cryptographically sign secrets, such as low-code development platforms.

New Solution Templates

New solutions this month include a Facilities Work Orders and a Care Facilities Survey template.
