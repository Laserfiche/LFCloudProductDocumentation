June 2023 Updates

The June 2023 update, released on June 1, 2023, includes the following new features.

Watch an overview of the new features here:

Expanded Regional Settings Support

Laserfiche Cloud has introduced new regional settings enhancements, including improved translations for Spanish and French, as well as the ability for administrators to control regional settings such as language, time zone, and date formats across their organization. These features have been released for the sign-in page, the app picker, the repository, public portal, direct share, the tasks inbox, forms submissions, and process reports, and will be expanded to administrative and process designer portions of Laserfiche Cloud in the future.

Administrators can set a default language and region for their organization in the Settings section of the Account Administration page. In addition, users can configure their own language and region settings in My Account.

Public Portal Update

Public Portal now follows your account-configured repository display name so that the repository name is displayed in a more friendly and understandable way.

Web Client Updates

URLs stored in text metadata fields will now be clickable, allowing users to easily navigate to linked pages directly.

The web client will now support streaming MKV files for Chrome & Edge. MKV files are a popular all-in-one media container that allows picture, audio, subtitle tracks, and image data to all be stored within a single file.

In-app Feedback Update

The In App Feedback tool, where Laserfiche users can share product feedback and suggestions, now supports the ability to attach files or images with feedback. This enables users to upload things like screenshots that help explain their comments, and enables the Laserfiche Product team to more clearly understand their intentions.

New Solution Templates

A new solution template, CSS/JavaScript/Formula Examples v2, is a collection of general CSS, JavaScript, and formula examples you can apply to customize your classic designer forms. To view the respective code or formula used for each example, click the “Show Code” button.

If you are interested in sharing your solutions with the wider Laserfiche community, please visit the Laserfiche Solution Marketplace and follow the Submit a Solution link at the top of the page.
