October 2023 Updates

The October 2023 update, released on October 5, 2023, includes the following new features.

Watch an overview of the new features here:

New Process Automation Capabilities

When designing a form, you can now customize the sub fields within an address field. For example, you could change the “Address Line 2” label to say “Apt., Unit, etc. (optional)” or remove the “Country” sub-field.

In Email Service Tasks, send an email to each member of an entire team at once by entering the team in the Send to field. For even more flexibility, you can dynamically email different teams by using a variable in this field that resolves to the name of a team.

In Workflow, the Search Repository activity can now search for documents with a specific tag applied to them. This provides yet another option for finding documents for Workflow to interact with. Additionally, the Set Records Management Properties activity can now automate the uncutoff action on records.

When configuring a lookup query rule, you can configure the maximum number of rows to return. This allows for better process performance for processes or forms using the rule.

Process Design Usability Enhancements

When editing a form or a business process, use the Manage rules link to go directly to the rules management page, allowing you to quickly navigate between where you are using the rule and the rule itself. If you create new rules or modify existing rules, you can then refresh the rule dropdown list directly to let your process or form use the updated rules.

When editing forms that contain a large number of fields, you can quickly find the field you want to edit by searching by the field name, variable name, or field ID. Click a search result to go directly to the field. Additionally, click the field rules link to directly open the field rules associated with that field.

It's also now easier to monitor running and completed workflows with new tooltips and it’s easier to see the total number of selected list items.

Form Scripting Updates

Designers can now bind to HTML events within custom HTML fields to run functions in the layout designer JavaScript sandbox. For example, create a custom button in one custom HTML field that starts a timer that is displayed in another custom HTML field.

The LFForm object can update more field settings such as address subfields, submission action buttons, pagination titles, previous and next buttons, signature and file upload button labels, and table column field settings. This gives you full control over all the labels in your form.

API Updates

The Repository Cloud API version 2 release brings new API functionalities such as large file upload, standardized long operation APIs, and many developer ergonomics improvements. A corresponding JavaScript API client library has also been released to make it easier for JavaScript developers to integrate with the Laserfiche repository.

Solution Templates

A variety of new solutions automate scanning, capture, and invoice processing with EzeScan. Additionally, a new solution streamlines department requests and tasks in Microsoft Planner.

If you are interested in sharing your solutions with the wider Laserfiche community, please visit the Laserfiche Solution Marketplace and follow the Submit a Solution link at the top of the page.
