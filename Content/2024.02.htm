February 2024 Updates

The February 2024 update, released on February 1, 2024, includes the following new features.

Watch an overview of the new features here:

Capture Enhancements

Capture Profile zones will now return a blank value when a zone's anchor is not found on the page, rather than capturing the text at the zone’s location. This behavior matches the expectation that if a field is not part of a document, its value will be blank. It also allows Workflow to respond to blank vs filled fields and process documents accordingly.

New Solution Templates

New solution templates include an example of a multi-step quiz and a workflow that can replace a Lookup Table's content with data from a spreadsheet.

Additionally, the new Point Submission & Redemption solution lets you efficiently administer a rewards program – e.g. a Customer Champions initiative – in which users can earn points for performing specified activities and then later redeem accumulated points for prizes.
