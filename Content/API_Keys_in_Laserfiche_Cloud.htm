API Key Overview

Self-hosted customers can now register, delete, or rotate API keys in Laserfiche Cloud to utilize their Laserfiche Direct Share integration.

Register a New API Key

API keys can be registered and maintained in the Integration Configuration section of your account Settings.

  1. Sign in to Laserfiche Cloud and navigate to Account Administration.
  2. In the left pane, click Settings.
  3. Click the Integration Configuration tab.
  4. Click the Register new API key button
  5. In the API Key dialog box, your key will be automatically created. To save the key, click Download .txt file or Copy to clipboard.

Note: Once you open the API Key dialog box, you can only view the key once. You will not be able to recover the key later. However, you can recreate or rotate the key at any time.

Delete or Rotate your API Key

If you no longer need your API key or would like to regenerate it, you may delete it by following the steps below:

  1. In Account Administration, click Settings in the left pane.
  2. Click the Integration Configuration tab.
  3. Click the delete link located in the top-right corner of your Delete API key dialog box.
  4. After deleting your available API key(s), you are able to register a new key if needed.
