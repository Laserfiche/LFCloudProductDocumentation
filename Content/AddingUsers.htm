Adding Users

Create user accounts that will be able to access services.

Creating Users in Laserfiche

  1. In the left pane, under Account Administration, click the Users tab.
  2. Click the Add User button as shown in the image below.
  3. In the Email field, enter the user's email address. By default, the Send email verification check-box is checked. Once this user is created, a verification email is automatically sent to the user to activate their account.

    4. Note: The verification link contained within the verification email will expire in 24 hours.

  4. Click the Check Email button to verify this email address is available. If this email address belongs to an existing user, an error message will appear, as seen in the image below.
  5. In the User Details section, enter user's information.

    6. Note: If you have created Groups, click the Group field. Choose a Group name from the drop-down menu. This will add this user to a group. To learn more about creating groups and adding users to groups, navigate to Creating Groups.

  6. In the Password section, choose between one of these two options:
    1. Send a password setup email: Selecting this option will gray out the Password and Confirm password fields. An email is sent to the user, prompting them to set the password for their account.
    2. Set initial password: Selecting this option requires filling out the Password and Confirm password fields. To prompt the user to set their password upon signing-in, click the check-box for User must change password on next sign-in.
  7. Optional: Turn on Multi-factor Authentication (MFA) to require that the user specify a second piece of information in addition to their password when signing in. By default, the Send multi-factor authentication code by email is checked.

    8. Note: Multi-factor authentication (MFA) is not supported when signing in through the Microsoft Office Integration prior to version 10.4.2.

  8. In the User Security and Access section, choose the User License Type for this user.

    Note: To learn more about License types, navigate to User License Types.

  9. In the Access Rights section:
    1. Presets: Use this drop-down menu to choose one of the following options: 
      1. Full Rights: Selecting this option allows the user full access rights to , , and . In addition to this, the user will have rights to grant themselves full access in the .
      2. Inherit All: Selecting this option means the user's access rights will be inherited from the user's group membership.
      3. Remove All Rights: Selecting this option means the user will not have any access rights to the and . User will not have any access rights to and unless inherited by the user's group membership.
