Scanning in Basic Mode

Basic Mode is one of two modes. It provides a fast and streamlined interface and is ideal for simple scanning tasks.

Basic scanning mode offers:

  • Basic image enhancements, such as page rotation, deskewing, and despeckling.
  • Text generation using OCR (optical character recognition).
  • An interface to fill in properties of the document before storing it.
  • The ability to send one document at a time to your repository.

Tip: If you need more advanced image enhancements, or want to scan multiple documents at once, you can use standard scanning mode.
