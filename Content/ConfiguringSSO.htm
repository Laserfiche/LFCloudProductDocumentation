Configuring for Single Sign-On (SSO)

  1. Sign in to and navigate to Account Administration.
  2. In the left pane, click Settings.
  3. Click the Single Sign-On tab. If you have not previously configured single sign-on with an identity provider, this page will look like the image below:

  4. Click Service Provider information to view the data required to configure the  SSO service.
  5. For step-by-step identity provider configuration options, choose an option below:
    • SAML Identity Providers: 
      • ()
  6. Once single sign-on is configured, you must select which users will sign in to using single sign-on.
    1. Under Account Administration, click the Users tab.
    2. Click the name of the user for whom you wish to turn on single sign-on.
    3. Check the check-box for Enable Single Sign-On.
    4. Click Save to finish.

      Note: To turn on single sign on for multiple people, you must predefine a list of accounts. To learn more, see Batch Importing Of User Accounts With a Comma Separated Values (CSV) File.
