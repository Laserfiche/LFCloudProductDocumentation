Configuring Columns

Columns in the Folder Browser provide information about the documents and folders in your repository, such as the creation date, template and field information, number of pages, and so on. You can choose which columns to display to show exactly the information that you need.

To configure columns in the Folder Browser, click Column Display and select Column Picker. In the Configure Column Display dialog box, under Available, double-click the columns you want to display. To stop displaying a column, double-click the column under Selected to remove it. You can also reorder columns by dragging and dropping them under Selected. Click Apply to save your changes. You can also reorder columns in the Folder Browser by dragging and dropping them.

List of Columns

The following table lists all of the standard columns, along with descriptions of the type of information that each column provides when displayed in the Folder Browser.

Columns in the Folder Browser
Column Description
Checked out by The user who currently has the document checked out, if it has been checked out
Created by The user who created the document or folder

Creation date

The date and time a document or folder was created.
Electronic file last modified The date and time the electronic file portion of a document was last modified
Entry ID The document or folder's entry ID
Extension The extension of the electronic file associated with the electronic document.
Has tags Whether or not the document or folder has tags applied to it
Indexed Whether or not the document has been indexed by the search service

Last nodified

The date and time a document or folder was last modified.
Last modified by The user who last modified the document or folder
Latest version The number of the most recent version of a document, if it is under version control
Location description The description of the document's location, if a location has been set for the document
Name The name of the document or folder. This column cannot be hidden

Pages

The number of pages in a document.

Template

The name of the template associated with a document or folder.
Text pages Whether text has been generated for the document. Values are Some, All, and None
Total document size The total size of a document, including electronic file, image pages, text, word location data, and thumbnails

In addition, all fields can be added as columns. Individual fields can be found under their template name or under All Fields. You can also add all fields in a template at once by double-clicking the template name.
