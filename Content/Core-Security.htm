Core Security: Entry Access Rights, Feature Rights, and Privileges

Three security features must be considered for all repositories. These features form the backbone of repository security:

  • Entry Access Rights, which control whether and how users work with documents, folders, and shortcuts in the repository.

  • Feature Rights, which control what tasks users can do across an entire repository.

  • Privileges, which provide access to powerful administrative features.

Other security features are considered supplemental security, and may or may not be necessary for any given organization.
