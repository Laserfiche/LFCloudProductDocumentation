Create a Application in

Follow the following steps to add an application to your  tenant: 

  1. In the Azure portal, on the left navigation panel, select Azure Active Directory.
  2. In the pane, select Enterprise applications. The All applications pane opens and displays a random sample of the applications in your tenant.
  3. In the Enterprise applications pane, select New application.
  4. Click Non-gallery application and follow the instructions to create a application
  5. When finished, navigate to one of the links below for additional information: 
    • Configure app properties
    • Assign users to an app
