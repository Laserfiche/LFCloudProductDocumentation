Team Filters

You can automatically assign tasks to subsets of teams with a User Task. By adding a filter to the team, you can assign the task to users who have a specific role on the team or who share some other criteria. A couple common filters are provided for you, or you can write a custom filter with expressions.

To filter team members

  1. In the Assign tab of the task, select the Team option.
  2. Chose the team you want to filter from the drop-down menu.
  3. Click the Add filter link.
  4. From the Select filter drop-down menu, select one of the following:
    • Managers Only: Click this option to assign the task to all the managers on the team. If only one user on the team is a manager, the user will be assigned the task directly and it will appear in their My Tasks tab. If there are multiple managers on the team, the task will appear in the Available tab for all managers. From there, they can assign the task to themselves to complete it.
    • Non-Managers Only: Click this option to assign the task to everyone on the team who is not a manager. The task will appear in the Available tab for all non-managers. From there, a team member can assign the task to themselves to complete it or a manager can assign the task to them. If only one user on the team is not a manager, the user will be assigned the task directly and it will appear in their My Tasks tab.
    • Select a saved filter for the team: If someone has already configured filters for this team, they will appear in a list here. Select a filter to apply it. If you want to make changes to this filter, click the Edit button in the top right of the filter's definition, and then follow the instructions for creating a filter below.
    • Create a new filter: Select this option if you want to create a custom filter with .

To create a custom filter

  1. Select Create new filter from the filter drop-down menu.
  2. Next to Name, type a descriptive name for your filter.
  3. Under Definition, select or . filters are easier to configure than filters because you can configure them by selecting a series of options from drop-down menus. The option allows for more customization and flexibility, but they can be harder to write, even with the pre-configured expressions provided.
  4. If you selected , click in the textbox.
    1. From the first drop-down menu in the dialog box that appears, chose if you want the filter to assign tasks to managers, non-managers, or users that have (or don't have) a specific role.
      • If you selected Role, choose = if the task should go to users with a specific role, or != if the task should go to users without a specific role. In the last drop-down menu, select a role.
    2. Once you've configured your initial condition, click Insert. The condition will appear in the text box below.
    3. Continue to add conditions as desired, using the and, or, and parentheses ( ) operators to combine conditions.

      4. Example: You could create a filter that assigns tasks to non-managers with the Quality Assurance role.

      Or, you could create a filter that assigns tasks to managers who don't have the Executive role.

  5. Use the Method drop-down menu to select methods that will filter the team members. Learn more. Ensure your finished script includes an expression that sets the value of the $result variable (which is declared before your script runs).
  6. After you define a filter, click the Test button to see how many users the filter will return. If the filter doesn't return any team members, edit the filter until it returns team members or add team members that fit your filter's criteria.
