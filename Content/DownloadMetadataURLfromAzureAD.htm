Download Metadata URL from

Follow the steps below to download App Federation Metadata URL in : 

  1. In the portal, select Enterprise applications.
  2. Find and select the App.
  3. In the Manage section, select Single sign-on to open the Single sign-on pane for editing.
  4. In the Signing Certificate section, find the App Federation Metadata Url and copy it.

Note: Navigate to Import Metadata in for the next step.
