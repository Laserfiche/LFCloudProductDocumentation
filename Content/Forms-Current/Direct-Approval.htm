Direct Approval

The Direct Approval option for a user task lets you perform actions without opening the task. Direct approval is an option you can select when you are configuring a user task. If a task has direct approval, you can complete the task through the details pane in Tasks on the Laserfiche home screen. Because you can perform direct approval actions on a form that has required fields, ensure that the direct approval option is disabled when you want a user to complete certain fields.

Using Direct Approval in the Tasks Tab

After the Direct Approval option is selected in a user task, the details pane in the Tasks tab for the user task displays action buttons based on the action buttons you configure in the task.
