Getting Started with Laserfiche Account Administration

Welcome to Laserfiche Account Administration. This page provides an introduction to Laserfiche Account Administration.

What is Laserfiche Account Administration?

Laserfiche Account Administration is your first step towards preparing your Laserfiche system. From here, you can view your Laserfiche Cloud billing subscription. It is also your first stop to configuring user accounts for authentication into the various Laserfiche services.

As a Solution Provider, Account Administration also allows you to create Provisional Accounts for prospective customers.

Accessing Laserfiche Account Administration

Upon joining Laserfiche Cloud, you will receive the following pieces of information:

  • Account ID
  • Administrator user name
  • Administrator password

Note: This user name grants access to the Laserfiche Cloud Account Administration site. You cannot use this account to sign in to view Laserfiche documents or view the customer listing page. Create and sign in using a user account to view the customer listing and create Laserfiche Cloud Provisional Accounts.

Visit https://signin.laserfiche.com/ and sign in using the aforementioned account ID and administrator user name and password.

View the Users and Groups tabs to manage accounts that can sign in and access your Laserfiche documents.

  1. Click Add User or Add Group to create accounts that will be able to access Laserfiche Cloud services such as viewing documents.
  2. Note the security options available for the account.
    • Set Laserfiche Privileges will allow the account to manage security settings for the Laserfiche repository through Repository Administration. An account cannot set Laserfiche privileges for itself.
    • Audit Reporting will allow the account to generate and view audit reports on the Laserfiche repository.
    • Access to Repository will allow the account to sign in to the Laserfiche repository.
    Note: The Inherited option indicates that the associated security setting will take its value from the account's group membership.

Note: You can now sign out of Laserfiche Account Administration and sign in using the user name that you just created to view documents in Laserfiche.

Important: At least one account should have Set Laserfiche Privileges in order to assign rights within Laserfiche Repository Administration.

View the Administrative Accounts tab to manage accounts that can sign in and access Laserfiche Account Administration.

  1. Click to create administrators that will be able to sign in and create users and groups within the Laserfiche Account Administration site.

Creating Provisional Accounts

Signing in as a user account takes you to your customer listing. Use the button to create Provisional Accounts.

In the listing, you can click on the Name or Account ID to edit customer information.

Click on the More Options button () and click Manage to create user accounts to allow the prospective customer to sign in.
