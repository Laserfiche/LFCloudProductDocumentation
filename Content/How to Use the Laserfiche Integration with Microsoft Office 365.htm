Working with the Laserfiche Integration with Microsoft Office 365

Before using the Laserfiche Integration with Microsoft Office 365 for the first time, you will need to sign in with your Microsoft Office 365 credentials the first time you attempt to edit a document. However, using the Laserfiche Cloud integration does not require any additional configuration.

To use the Laserfiche Integration with Office 365:

  1. Sign in to Laserfiche Cloud and navigate to your repository by clicking on the App Picker icon in the upper-right and choosing the repository you want to work in.
  2. Once in your repository, upload a new Word, Excel or Power point Document or select one of the existing documents in your repository.
  3. Click on the document title to open the document preview screen.
  4. Click the Edit With button in the top right. To keep working within Laserfiche Cloud client, choose the Microsoft Word Online option. This will redirect you to the Microsoft's hosted log-in page for Office 365.

    5. Note: Choosing Microsoft Word will prompt you to open the Laserfiche Plugin Connector, which will download the file locally to edit on your desktop.

  5. Complete signing in to Microsoft. After signing in, you will be redirected to Laserfiche with the document open through Office Online and available for editing.
  6. Another user may co-author the document at the same time as you by signing in to Laserfiche Cloud and Microsoft Word Online. Both you and the other user will see the cursor location of the other user. All changes from both users will be persistently saved to document within the repository.

File size limits in Microsoft Office 365 online:

There are file size constraints when viewing or editing documents with the Laserfiche Integration with Office 365. This is due to the file size limits from Microsoft Office Online.

Microsoft 365 Online Application Mode Size Limit Notes
Excel for the web Edit 25 MB   
Excel for the web View 25 MB   
PowerPoint for the web Edit 300 MB  Subject to overall 60 second time out for file downloads. In addition, for embedded media, the size limit is 100MB for most embedded media types. For WAV files the limit is 100KB.
PowerPoint for the web View See notes Subject to overall 60 second time out for file downloads.
Word Edit 100 MB  
Word View See notes Subject to overall 60 second time out for file downloads.

The above values are taken from Microsoft's documentation.

To enable the Laserfiche Integration with Office 365:

  1. Click the App Picker icon in the upper-right of Laserfiche Cloud.
  2. Select Repository Administration.
  3. On the Management tab, open the Integrations panel.
  4. Go to the Services tab and set the Edit with Office Online toggle to ON.
  5. Click Save to save your changes.

To disable the Laserfiche Integration with Office 365:

  1. Click the App Picker icon in the upper-right of Laserfiche Cloud.
  2. Select Repository Administration.
  3. On the Management tab, open the Integrations panel.
  4. Go to the Services tab and set the Edit with Office Online toggle to OFF.
  5. Click Save to save your changes.

Working with Legacy File Formats

Microsoft Office Online does not support editing files in binary formats such as doc, ppt, and xls, directly. You can choose to allow Office Online to convert those documents to modern formats like docx, pptx, and xlsx to edit them in Office Online.

However, the conversion process is not always perfect. Be aware that the conversion process can introduce changes to the layout and formatting of the document and that the conversion process may not support all legacy options. For example, legacy format .doc files with digital signatures may not be converted properly.
