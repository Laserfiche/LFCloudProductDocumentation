Importing a Briefcase

A briefcase is a special collection of Laserfiche documents, including their metadata, that can be exported and re-imported into your repository. Briefcases are useful for sharing documents with other sites that use Laserfiche, or for backing up a set of documents. A briefcase includes not only the documents themselves, but also their folder structure, and Laserfiche-specific information such as annotations and metadata.

Briefcases are saved as files with the extension LFB.

Importing with Drag and Drop

  1. Locate the briefcase you want to import in Windows.
  2. Open Laserfiche and navigate to the folder into which you want to import the briefcase.
  3. Drag the briefcase from Windows into your folder. A Drop files here box will appear.
  4. Drop the briefcase into the box.The Import dialog box will open.
  5. If the briefcase was encrypted, you will be prompted to provide the encryption password.
  6. To import the briefcase to a different folder than you have open, type or browse to a new folder name in the Folder option.
  7. If you want to merge folders with the same name in the destination folder, select Merge with existing folder structure. If you want to instead rename the imported folders, clear this option.
  8. If the briefcase contains documents with templates, the Select Template dialog will open, listing the templates in the briefcase. If any of these templates are a perfect match for a template already in the repository, you can choose to match the template by selecting the appropriate template under Select matched template. Regardless of whether the templates match, you can choose to add the template, or to select Ignore fields to ignore it and drop its field information from documents in the briefcase.
  9. Click OK to continue. The briefcase will be imported.

Importing with the Import Command

  1. Open Laserfiche and navigate to the folder into which you want to import a file.
  2. Click the Import button (). The Import Files dialog box will open.
  3. Click Upload.
  4. Select a briefcase to import and click Open.
  5. Once the briefcase has finished uploading, click OK. The Import dialog box will open.
  6. To import the briefcase to a different folder than you have open, type or browse to a new folder name in the Folder option.
  7. If you want to merge folders with the same name in the destination folder, select Merge with existing folder structure. If you want to instead rename the imported folders, clear this option. Click OK to continue.
  8. If the briefcase contains documents with templates, the Select Template dialog will open, listing the templates in the briefcase. If any of these templates are a perfect match for a template already in the repository, you can choose to match the template by selecting the appropriate template under Select matched template. Regardless of whether the templates match, you can choose to add the template, or to select Ignore fields to ignore it and drop its field information from documents in the briefcase.
  9. Click OK to continue. The briefcase will be imported.
