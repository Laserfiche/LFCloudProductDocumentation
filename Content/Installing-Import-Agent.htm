Installing Import Agent

This section contains the information you'll need to successfully install Laserfiche Import Agent and Laserfiche Email Archive.

System Requirements

The system requirements for the computer hosting Import Agent are listed below.

  • CPU:
    • Minimum: 1.8 GHz processor
    • Recommended: Quad-core 2.8 GHz processor or faster
  • Memory:
    • Minimum: 1 GB RAM
    • Recommended: 2 GB RAM or more
  • Operating System:
    • Windows Server 2019
    • Windows Server 2016
    • Windows Server 2012 R2
    • Windows Server 2012
    • Windows 10
    • Windows 8.1

Software Requirements

  • .NET Framework 4.7.2 or later
  • Microsoft Visual C++ 2012 Redistributable (x64) - 11.0.61030
  • Microsoft Visual C++ 2012 Redistributable (x86) - 11.0.61030
  • Microsoft Visual C++ Redistributable for Visual Studio 2017 (x64) -14.11.25325
  • Microsoft Visual C++ Redistributable for Visual Studio 2017 (x86) -14.11.25325

Installing Import Agent

The Import Agent installation package can be obtained from your reseller, or, if you have a Laserfiche support site account with sufficient privileges, downloaded from the support site. Select the most recent version of the installation package. In addition to the Import Agent install files, it will contain a license for use when registering Import Agent.

To install Import Agent

  1. Open the Import Agent install package and launch Autorun.exe.
  2. Select Install Laserfiche Import Agent.
  3. Select which language you want to run the installation in.
  4. The Laserfiche Import Agent setup will launch and the Welcome screen will appear. Click Next.
  5. In the End-User License Agreement step, read the license agreement and select I accept the terms in the License Agreement if you agree to the terms of the license agreement. Click Next.
  6. On the Custom Setup step, select whether you want to also install Laserfiche Email Archive and a local instance of the OCR engine. Modify the Install directory option if you want to change the default installation folder. By default, Import Agent is installed in "C:\Program Files\Laserfiche\Import Agent." Click Next to continue.
  7. On the Laserfiche Product Activation step, select the Activate using license file option and browse to the path where the Import Agent license file can be found. By default, this is in the License subfolder of the folder where you extracted the installation files.
  8. On the Laserfiche Service Settings step, specify authentication information for the Import Agent service and Laserfiche Email Archive service. By default, the services will run as the Local System account. You do not need to specify a password for the Local System account. Select the Create Windows Firewall exception option for the installation to create a Windows Firewall exception for the ports used by the Laserfiche Email Archive service (if you do not do this, you will need to configure the ports manually).
  9. On the Ready to Install step, click Install to begin the installation process.
  10. The Prerequisites step may appear if Import Agent needs to install additional components. A green check will appear next to installed items and a red X will appear next to items that have not been installed. If one or more of the items on the list has a red X next to it, install that item and then click Retry. Click Next.

    Important: Bypassing prerequisites may cause Import Agent to function incorrectly and is not recommended.

  11. On the Installing Laserfiche Import Agent step, click Next when the installation finishes.
  12. Click Finish to exit the setup wizard.

Laserfiche Import Agent Service

Import Agent uses a Windows service named Laserfiche Import Agent Service to retrieve, import, move, and delete files in Windows folders. In some cases, you may need to modify the Windows account associated with this service.

To configure Windows rights for the Laserfiche Import Agent service

  1. Open the Windows Services.
    1. Open Windows Control Panel. Depending on the version of Windows either switch to Classic View or select to view by Large icons or Small icons.
    2. Open Administrative Tools.
    3. Open Services.
  2. Double-click Laserfiche Import Agent Service.
  3. Select the Log On tab.
  4. Select one of the following options:
    • Local System account. Only select this option if Import Agent is installed on the same machine that hosts the files you want to import and Import Agent is not configured to use Windows authentication to sign into the repository.
    • This account. Select this option if Import Agent will access files across a network and/or Import Agent is configured to use Windows authentication to sign into the repository. Specify a Windows user account and password. Keep in mind the following considerations:
      • If the password associated with the account changes, you may have to manually update this dialog box.
      • If Import Agent is configured to use Windows authentication to sign into the repository, the Windows account specified here will be used. Ensure this account has sufficient Laserfiche rights to perform all the tasks required by Import Agent.
      • Ensure this Windows account has sufficient Windows rights to retrieve, import, move, and/or delete files in Windows directories used by Import Agent.
  5. Click OK.

To change the status of the Laserfiche Import Agent service

  1. Open the Import Agent Configuration Utility.
  2. Select Service from the menu.
  3. Select one of the following options:
    • Start
    • Stop
    • Pause

      • Tip: Choosing Stop or Pause disables all profiles.
