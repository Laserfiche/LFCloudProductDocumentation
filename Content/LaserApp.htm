Saving Laser App Forms to Laserfiche

Laser App is a third-party application that helps you fill out and file forms. Using the Laserfiche integration with Laser App, you can then store those forms in Laserfiche, and extract information from your forms to populate Laserfiche metadata.

The steps necessary to configure the integration depend on whether you are using Laser App Enterprise or Laser App Anywhere. See Configuring and Using the Laser App Enterprise Integration or Configuring and Using the Laser App Anywhere Integration for more information.
