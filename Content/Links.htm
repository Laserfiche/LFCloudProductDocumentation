Links

A link is a metadata type in Laserfiche that allows you to connect two related documents or folders, such as an email and an document. Links use a word or phrase of description to help you categorize your documents and keep track of related documents. For example, you might connect documents of different types that are related in some way, such as an agenda and its packet or an email message and its document. You might also use document relationships to indicate document status, such as a document that has been superseded and the document that superseded it.

Document Links

Viewing, Adding, and Removing Links

You can view the current links to a document, add links, or remove links from the right pane of both the Document Viewer and the Folder Browser.

  1. Select a document or folder in the Folder Browser, or open a document in the Document Viewer.
  2. In the right pane, select the Details tab. Any current links will be displayed under Links.
  3. To open a linked document or folder, click its name. It will immediately open in a new
  4. To create a link, remove a link, or view link information, click View Links. All existing links and their relationship types and paths will be displayed. You can then do one of the following:
    • To create a link between this document or folder and another, click Create link. Specify the Relationship type and then navigate to a document or folder. Click Save to save the link.
    • To remove a link between this document or folder and another, click the delete button (X) next to the linked document.

Creating and Deleting Relationship Types

You can create and delete relationship types that will be used to link documents in Repository Administration.

Creating a Relationship Type

  1. Open repository management by selecting the app picker and then selecting Repository Administration.
  2. On the left, under Metadata, select Document Relationships.
  3. Click the Add button () to add a new relationship type. Alternately, click the name of an existing relationship type to modify it.
  4. In the Relationship option, type a name for the relationship from the source document to the target document.
  5. In the Reverse relationship option, type a name for the relationship from the target document to the source document.
  6. Optional: In the Description option, type a description for the relationship.
  7. Click OK to save the relationship.

Deleting a Relationship Type

To delete a tag in Repository Administration, open Document Relationships, select the relationship you want to remove, and click the Delete button (). This will permanently remove the relationship. The relationship type will also be removed from all documents and folders that contained it.
