Using the Laserfiche Integration with Microsoft Teams in the Microsoft Teams App

The Laserfiche Integration with Microsoft Teams can be used from the Microsoft Teams app much the same as it can be from the Microsoft Teams desktop or browser application. However, some of options are accessed in slightly different ways.

Navigating Laserfiche in Teams

To open the Laserfiche Integration, click the More button and then select the Laserfiche icon.

You'll be immediately prompted to sign in to your Laserfiche Cloud account either from the Home or Files tabs in the app. You will only need to sign in to one of these spots to gain access to the app.

The components of the Laserfiche app in Microsoft Teams:

  • Home: In this tab, press Save recent files to Laserfiche to select files shared in your chats or channels that you want to save to Laserfiche. The first time you save files to Laserfiche, press the Authorize button to give Laserfiche permission to access files stored in Microsoft Teams. You can also sign out of Laserfiche from this tab.
  • File: Click Open file, and navigate to the file you want to open. Then, select the file, and press Open. You can interact with the file just as if you had opened it in Laserfiche by adding annotations, metadata, and more. Note that only individual files, not shortcuts or folders, can be opened and shared with the Laserfiche.
  • Click the Bot button in the upper right to explore what you can do with Laserfiche in Microsoft Teams. Type Sign In or Sign Out to sign in and out of Laserfiche. Type Help to see a summary of what you can do with the Laserfiche app in Microsoft Teams.
  • Use the More menu to view the Laserfiche app's profile, to find entries from the Laserfiche app in the chat, and to mute the Laserfiche app in the chat.

Sharing Files Stored in

From a Microsoft Teams chat or channel

  1. Press plus button next to the new message text box, and press the Laserfiche icon.
  2. Select Share Laserfiche files.
  3. Press Add file to select files to add to a Share Files card.
  4. Navigate to the files you want to share, and select them. Press Select.

    5. Note: Folder and shortcut files cannot be shared.

  5. Review the files you chose to share.
    • Add a comment below each file to provide a description of the file and give any instructions or notes associated with the file.
    • Select Hide file information to hide the file's name for security purposes. You may want to hide the file's name if you're sharing it in a channel or chat where only some of the participants should have access to the file or even know the file's name. Users will need to sign in with their Laserfiche credentials to view the file's name.
    • Use the trash can button to remove files from the card.
    • Click the Add file button to add more files to the card.
  6. Press Preview & Send.
  7. To edit the Share Files card, press Edit. Otherwise press Send to send the card in the chat or channel.
  8. A Share File card is created and automatically added to your message. Press any of the files listed to open the file in Teams.

Opening Laserfiche Files in Microsoft Teams

You can interact with files just like you can in Laserfiche by applying annotations, editing metadata, and more. Learn about working with Laserfiche files.

From a Share File card in a chat or channel

  • Press any of the files listed to open the file in Teams.

From the Files tab

  1. Press Open file.
  2. Navigate to the file in your repository, and press Open.

After importing files to Laserfiche

  • If only one file was imported, press the View saved file button.
  • Press Send saved file to the Laserfiche Chat tab, and then press the file in the Imported File card.

Saving Files to

From a Microsoft Teams chat or channel

  1. Press plus button next to the new message text box, and press the Laserfiche icon.
  2. Select Save files to Laserfiche.
  3. Select the files you want to save to Laserfiche, and press Select.
  4. Use the Browse button to navigate to where you want to save the files in the repository. Then press, Select.
  5. Press Save.
  6. The files will go through the Laserfiche import process. Edit the file's name, location, and metadata as desired. Press Import.
  7. Press Send file to the Laserfiche chat tab to send the imported files in Imported File cards to a Laserfiche chat thread. If you only imported one file, you can press the View saved file button to open the file.

From a file attached to a message in a Microsoft Teams chat or channel - This doesn't seem to be available in the app (at least not yet)

  1. Click the More options ellipses at the top right of the card.
  2. Point to More options in the menu, and select Save files to Laserfiche.

  3. Select the files you want to save to Laserfiche, and click Select. If the message only has one file, this step will be skipped.
  4. Use the Browse button to navigate to where you want to save the files in the repository. Then click, Select.
  5. Click Save.
  6. The files will go through the Laserfiche import process. Edit the file's name, location, and metadata as desired. Click Import.
  7. Click Send file to the Laserfiche chat tab to send the imported files in Imported File cards to the Laserfiche Chat tab. If you only imported one file, you can click the View saved file button to open the file.
