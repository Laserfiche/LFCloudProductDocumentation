Document Properties

You can view a document's properties to see valuable information about the document.

  • If using an Android device, long press the document. If using an Apple device, swipe left on the document.Tap Properties.
  • If the document is already open, tap the More button in the top right, and tap Properties.

Geotag

If a document is created from a photo taken on the mobile device and if the geotag option has been configured by an adminstrator, the Geotag property provides the GPS coordinates for where the document was taken. If the document is composed of multiple photos taken in different locations, the coordinates for the first photo in the document will be recorded. Tap the Open in map application link to plot the photo's location on a map. Depending on your device's settings and applications, you may be prompted to choose a map application first.

General Properties

  • Location: The document's path in Laserfiche.
  • Date created: The date and time the document was added to Laserfiche.
  • Last modified: The date and time the document was last changed.
  • Created by: The account name of the person who added the document to Laserfiche.
  • Owner: The document's owner has special security rights to the document. Usually, the person who creates the entry is its owner.
  • Entry ID: The unique identifying number assigned to the document.
  • Number of pages: How many pages are in the document.
  • Indexed: Whether the document has been indexed or not. Documents must be indexed for their text to be searchable.
