Emailing Links to Documents and Folders

You can email mobile URLs to documents and folders that other users can open using Laserfiche Mobile and Laserfiche.

To email a link to a document or folder

  1. If using an Android device, long press the entry you want to email a link to. If using an Apple device, swipe left on the entry you want to email a link to. If you want to email links to multiple documents or folders, tap the checkboxes next to the entries and tap the More button in the top right.

    2. Tip: If you already have the document that you want to send a link to open, tap the More button in the top right.

  2. Tap Email Link.
  3. A list of the apps on your phone that can send email might appear. Optionally, before selecting an email app, select if you want to use this app by default each time you email a link to a mobile document. Tap the email app you want to use.
  4. The email application will open and an email will be populated containing one or more links. The links the email contains depends on what has been configured by your administrator. The email can be configured to have:
    • A link for opening the document or folder using Laserfiche, beginning with http://.
    • A link for opening the document or folder using Laserfiche Mobile, beginning with lfmobile://.
    • Links to download the app in the Google Play or iTunes App Store.
  5. Edit the email as desired.
  6. Send the email.

Note: By default, when sending a document URL, the URL in the email will be configured to display imaged pages if any are available. If no imaged pages are available, the electronic document will open in your device's default application for that type of document. PDFs will open in the Laserfiche Mobile app. A document's text will only open when it’s the only thing that exists for that document. If using an Android device, you will be promoted to choose an application to open the documents with PDFs.
