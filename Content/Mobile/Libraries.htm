Libraries

Libraries let you group related documents and folders so they can be quickly viewed together, even when they are located in different areas. Libraries are unique to each user. Users can customize their libraries however they need to without affecting the actual organization of the system.

Example: Chris is a professor who needs easy access to a variety of documents to teach his classes, such as instructions to set up a projector in a classroom, the latest campus construction update, the student disciplinary policy, and his current syllabuses. These resources are stored in many different folders, but to allow for easy access, Chris creates a Resources library. He puts any documents or folders he wants quick access to into this library and removes them from the library when they are no longer relevant. Now when he has trouble setting up a projector in his classroom, he can easily pull up the instructions by opening the Resources library.

Before you can add entries to a library, you must first create the library.

Tip: If you are accessing your system from multiple devices, tap the refresh button to keep your libraries synchronized.

Create a library

  1. In the Folder Browser, swipe right and tap Libraries. Or, tap the main menu button in the top left, and tap Libraries.
  2. Tap the Add button in the top right.
  3. In the New Library dialog box, enter a name for the library.
  4. When finished, tap OK in the dialog box.

Add a document or folder to a library

  • If using an Android device, long press the document or folder and tap Add to Library. If using an Apple device, swipe left on the document or folder and tap Add to Library. Select the desired library and tap Add.
  • To add an open document to a library, tap the More button in the top right and select Add to Library. Select the desired library and tap Add.

Add multiple documents and/or folders to a library

  1. Browse to the documents or folders you want to add to a library.
  2. Tap the gray checkbox to the left of each document or folder you want to add. When finished, tap the More button in the top right.
  3. Tap Add to Library.
  4. In the Libraries dialog box, select the library you want to add the document or folder to.

    5. Tip: You can create a new library at this point by tapping the Add button in the top right. Type a name for the library in the New Library dialog box, and tap OK.

  5. Tap Addat the bottom.

Open a library

  1. In the Folder Browser, swipe right and tap Libraries. Or, tap the main menu button in the top left, and tap Libraries.
  2. Tap the library you want to open. Tap any document or folder in the library to open it.

Rename a library

  1. In the Folder Browser, swipe right and tap Libraries. Or, tap the main menu button in the top left, and tap Libraries.
  2. If using an Android device, long press the library you want to rename. If using an Apple device, swipe left on the library you want to rename.
  3. When prompted, tap Rename Library.
  4. Type a new name for the library.
  5. Tap OK.

Remove documents and/or folders from a library

  1. In the Folder Browser, swipe right and tap Libraries. Or, tap the main menu button in the top left, and tap Libraries
  2. Tap the library you want to remove documents and/or folders from.
  3. Tap the checkboxes next to the documents and/or folders you want to remove.
  4. Tap the More button in the top right.
  5. Tap Remove from Library.
  6. In the Confirm dialog box, tap OK to confirm removal of the entry, or tap Cancel to cancel.

Delete a library

  1. In the Folder Browser, swipe right and tap Libraries. Or, tap the main menu button in the top left, and tap Libraries.
  2. If using an Android device, long press the library you want to delete. If using an Apple device, swipe left on the library you want to delete.
  3. Tap Delete Library.
  4. When prompted to confirm the deletion, tap OK.
