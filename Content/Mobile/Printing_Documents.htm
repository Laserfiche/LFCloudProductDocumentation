Printing Documents

Documents can be printed using the Android or Apple app.

Apple

If you are using an Apple device and want to print Laserfiche image pages directly from the Laserfiche app, you can use the Print Image Pages option. Note that this option requires iOS 6.0 or higher and an AirPrint compatible printer. For more information, see KB article ht4356 at support.apple.com.

To print a Laserfiche image document

  1. Swipe left on the Laserfiche document you want to print. If you already have the document open that you want to print, tap the More button in the top right.
  2. Tap Print Image Pages.

    Note: You can tap Open In... to print the TIFF file. This option lets you print to any printer app.

  3. On the Page Range screen, select if you want to download and print all pages or use the scroll wheels to define a range of pages to download and print. Tap OK.
  4. Once the document is downloaded, the Printer Options page will appear.

    5. Note: You can cancel the download in progress by tapping Cancel. If you have the document open, you can dismiss the download dialog box, and let the download continue in the background, by tapping Hide. To show the dialog box again, tap the More button in the top right, and select Print Image Pages.

  5. Tap Select Printer and select the printer you want the document to print to. The printer you select must be an AirPrint compatible printer.
  6. Other printer properties will appear on the Printer Options page depending on what type of options are supported by your printer (e.g., number of copies, double-sided, etc.).
  7. When finished, tap Print. To cancel the print job, tap Cancel.

To print an electronic document

  1. Open the electronic document you want to print.
  2. Tap the More button in the top right.
  3. Tap Open In....
  4. Select the printer app and follow the app's instructions to print the electronic document.

Note: If the electronic document has Laserfiche image pages associated with it, you can use the Print Image Pages option to print those pages.

Android

If you are using an Android app, you can use the Open In... option to open the Laserfiche image pages in a printer app. Follow the instructions in the printer app to print the pages.

To print a Laserfiche image document or electronic document

  1. Open the document you want to print.
  2. Tap the More button in the top right.
  3. Tap Open In....
  4. Select the printer app and follow the app's instructions to print the electronic document.
