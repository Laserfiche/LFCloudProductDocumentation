Renaming Documents and Folders

To rename a document or folder

You can rename a document or folder using either of the following methods:

  1. If using an Android device, long press the document or folder. If using an Apple device, swipe left on the entry. Tap Rename.
  2. In the Rename dialog box, type a new name for the entry.
  3. Tap OK.

Alternatively

  1. Tap the checkbox next to the document or folder you want to rename.
  2. Tap the More button in the top right.
  3. Tap Rename.
  4. In the Rename dialog box, type a new name for the entry.
  5. Tap OK.
