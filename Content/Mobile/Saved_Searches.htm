Saved Searches

Although searches cannot be saved in Laserfiche Mobile, you can access and run saved searches that were saved in Laserfiche.

To run a saved search in Laserfiche Mobile

  1. If using an Android device, tap the search button in the top right. If using an Apple device, pull down to reveal a search box and tap inside the box.
  2. A list of searches saved within Laserfiche will be displayed under Saved Searches. Tap the saved search you want to run.
  3. To view a document or folder returned from a search, tap it.

Note: Saved searches can not be deleted from Laserfiche Mobile.
