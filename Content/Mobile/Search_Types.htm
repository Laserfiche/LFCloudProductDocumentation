Search Types

You can search document text, document or folder name, fields, or annotation text.

  1. Navigate to the folder you want to search in. If using an Android device, tap the search button in the top right. If using an Apple device, pull down to reveal a search box and tap inside the box.

    Note: The search will search the documents and folders inside the folder you are currently in.

  2. Tap the area below Search Typesand tap each search type you want to run. A check mark will appear next to the ones you select:
    • Document text: Searches the text of documents
    • Entry names: Searches the names of documents and folders
    • Fields: Searches the fields assigned to documents and folders
    • Annotation text: Searches the text of sticky note, text box, and callout text box annotations.

    Note: Although annotation text is searchable in Laserfiche Mobile, they are not displayed in the app. Open the document in the iOS app, Web Access, or the desktop Client to view annotations.

  3. Once you've selected the search types you want to search, tap the back button.
  4. Enter a search term or choose a recent search term from the Recent Search Terms list. Tap the search key on your device to perform a search.

    5. Note: To cancel a search in progress, tap Cancel.

  5. The top of the screen will display the search term. If you performed a text search, the results that match your search criteria will be returned (fifty at a time) as a list sorted by Laserfiche relevance ranking. Scroll to see more results. The text below each search result indicates the context of the search term.
  6. To view a document or folder returned from a search, tap it.

    7. Note: Tapping an imaged document will open the document in the Document Viewer to the page containing the first search hit. If the search term appears elsewhere in the text of the document, it will also be highlighted throughout the document's image.

  7. You can view a document or folder's fields. If using an Android device, long press the document or folder and tap Fields. If using an Apple device, swipe left on a document or folder and tap Fields. Learn More

Note: Advanced searches are not available in the Laserfiche Mobile user interface. However, you can run an advanced search that was saved in Laserfiche.
