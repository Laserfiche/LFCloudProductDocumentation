Settings

To access settings and other app information, tap the main menu button at the top left and tap the Settings button in the bottom left corner of the main menu.

Upload using Wi-Fi only

This option controls if new documents will be uploaded using Wi-Fi and cellular data or Wi-Fi only. By default, this option is turned off. When turned off, the app will use Wi-Fi when available, and cellular data when Wi-Fi is not available. If this option is turned on, only Wi-Fi will be used to upload documents (not cellular data).

Save to gallery

This option controls if photos you take to upload as pages of a new document will be saved to the local photo gallery of your device. By default, this option is turned off and photos will not be saved to the photo gallery. Make sure you have given Laserfiche Mobile the right to access your camera and photos.

Help

  • Tapping Help opens the help files for Laserfiche Mobile.

Answers

  • Tapping Answers will leave the Laserfiche Mobile app and take you to Laserfiche Answers.

Connection

  • Account ID: Displays the Laserfiche Cloud 9-digit account ID.
  • Repository: Displays the repository you are connected to.
  • User name: Displays the user currently logged into the repository.

Version

  • Application: Displays the current Laserfiche Mobile Application version.
  • Laserfiche Mobile Server: Displays the current Laserfiche Mobile Web Server version.
  • Laserfiche Server: Displays the current Laserfiche Server version.

Copyright

  • Displays the copyright information for the app.

Third-party add-ons:

  • Displays information about the add-ons used in Laserfiche mobile. Learn more.

Send Feedback

  • Tapping Send Feedback lets you send Laserfiche Mobile feedback via the device's native email app. The To field will be populated for you. Enter your name and organization, ensure your device model and OS are filled out correctly, type your feedback, and send the email.
