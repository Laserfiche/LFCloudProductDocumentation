Templates and Fields

You can add and edit templates and fields in Laserfiche Mobile, based on your Laserfiche security rights.

If a template has a color associated with it, a colored dot will appear next to the documents or folders the template is applied to.

To modify a document or folder's templates and fields

  1. If using an Android device, ,long press the document or folder. If using an Apple device, swipe left on the document or folder. Select Fields. If you are uploading a new document, if you have selected the checkbox next to a document or folder, or if you are editing an open document, tap the metadata button .
  2. If a template has been assigned to the document, it will be displayed. If no template has been assigned to the document, you will see No template assigned. Tap the template drop-down menu to select a template from the list and tap OK.
  3. Tapping Add/Remove Fields will display all available fields in Laserfiche. Select the checkboxes next to the fields you want to add to the document or folder, clear the checkboxes next to the ones you want to remove from the document or folder, and tap OK.
  4. Tap inside a field box to enter/edit a field value. If using an Apple device, tap the forward or back arrows above the keyboard to move between fields.
  5. When you've entered all desired field values, tap the check mark to save the field values. Alternatively, tap the cancel button to discard all changes.
