Viewing Document Text

If a document you are viewing has text and an image and/or electronic document associated with it, you can tap the More button in the top right, then tap Switch View to access these views. If the document has text but no image, the text will appear automatically.

When viewing text:

  • Scroll down to see all the text on a given page.
  • Swipe right to go the next page. Swipe left to go to the previous page.
  • To go to a specific page, tap the page number box , then tap the thumbnail for the page you want to view.

When you have finished looking at the document's text, you can:

  • Tap the More button to switch views, email a link, add the document to a library, view the document's properties, or export the electronic document portion of the document (if applicable).
  • Tap the metadata button to edit the document's fields.
  • Tap the annotation button to view annotations on the document.
  • Tap the back button to return to the document's folder.
  • If using an Android device or iPad, tap the close button to close the document.
