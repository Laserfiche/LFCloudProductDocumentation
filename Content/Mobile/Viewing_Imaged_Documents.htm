Viewing Laserfiche Imaged Documents

Once you have located an imaged document, tap it to open it.

If using an Android phone or iPhone, the document will open and fit the width of the screen.

If using an Android tablet or iPad, the document will open in the Document Viewer. To view the document full screen, tap the full screen button in the top right. When viewing the document in full screen mode, tap the metadata button ,or annotation button , to view that information to the right of the image.

To zoom in

  • Pinch outward to zoom in.
  • Double-tap, with one finger, on the specific portion of the document you want to zoom in to.

To zoom out

  • Pinch inward to zoom out of a document.
  • Double-tap a zoomed in document.

To navigate between pages

  • Swipe left to go the next page, and swipe right to go to the previous page.
  • To go to a specific page, tap the page number box, then tap the thumbnail for the page you want to view.

Options for interacting with the document

  • Tap the More button for a variety of options for interacting with the document such as switching from image view to text view and vice versa, printing its image pages (Apple only),emailing a link to it, adding it to a library, viewing its properties, or exporting the electronic document portion of the document (if applicable).
  • Tap the metadata button to edit the document's fields.
  • Tap the annotation button to view annotations on the document.
  • Tap the back button to return to the document's folder.
  • If using an Android device or iPad, tap the close button to close the document.

Options for interacting with the image

Tap the screen with one finger to see the following options along the bottom of the screen.

  • Rotate: Tap the rotate button to rotate the displayed image 90 degrees clockwise. You must rotate each page individually.
  • Reorder pages: Tap the two arrows to reorder the pages in the new document. In the Move current page to dialog box, specify which page the image will become. Type a number in the text box or tap the plus and minus buttons. Tap OK.
  • Delete: Tap the delete button to delete the page currently shown.
