Navigating the Repository

There are four major parts of the Laserfiche repository: the repository itself, which contains your documents and folders; the repository administration page, where you configure options such as users and groups, security, metadata, records management, and activity monitoring, among others; and audit trail, where you configure audit settings and run audit reports. You can also use additional tools to work with your repository contents, including the Laserfiche mobile app, capture tools such as Snapshot and Import Agent, and the public portal.

When you first sign in to Laserfiche, your starting page will depend on your account type, account settings, and the URL you used to access the repository. You can switch to other apps in Laserfiche using the app picker () in the upper right corner next to your user name. You can also change your default starting page; see Setting a Default Application for more information.

Laserfiche Repository

Your repository contains the documents and folders in Laserfiche. To open your repository, open the app picker and select your repository name in the Repository section.

The repository itself has two main sections: the Folder Browser and the Document Viewer.

Folder Browser

When you log in to the repository, you will begin in the Folder Browser. You can navigate your repository quickly using the folder tree on the left. To view the contents of a folder, you can select it and view the documents it contains in the center. You can also select individual documents and folders in the center. You can perform actions by clicking the buttons on the toolbar at the upper right, or by right-clicking a document or folder. (For instance, you can click the Scan button to scan a new document, or select a document and click the Download button, or right-click it and select Download, to download it.) On the right, you can view additional information (called metadata) about the selected document or folder.

You can also perform a quick search by typing search terms in the search bar at the top, or click Search Filters to configure a search using a wider variety of options.

Starred documents and folders, recently-opened documents, and the results of your most recent search can be accessed in the upper left corner; your recycle bin is in the lower left corner, beneath the folder tree.

For more information, see Adding Documents and Folders, Viewing Documents and Folders, and Organizing Documents and Folders.

Document Viewer

The Document Viewer allows you to view the pages, metadata, annotations, and other information about documents with image pages, including PDFs. You can also open Microsoft Word, Excel, or PowerPoint documents for viewing and editing, or stream video and audio files. (Other types of documents will be downloaded for opening in their native application by default.)

On the left, you can quickly scan through a document's pages using the thumbnails. To view a page, select it and it will display in the center. On the right, you can view additional information (called metadata) about the selected document or folder. The toolbar at the top allows you to perform actions such as zooming in or out, adding annotations, switching to full screen mode, or saving changes. Above that, the search bar allows you to quickly search within this document.

For more information, see Viewing Documents and Folders and Working with Document Contents.

Scanning

Laserfiche Scanning allows you to use a scanner to capture paper documents and store them in Laserfiche. You can also use Laserfiche Scanning to import images already on your computer. While scanning, you can perform image enhancements such as rotating or deskewing a page, generate text from the scanned documents, and store additional information about the documents along with them.

For more information, see Scanning.

Office Integration

Office Integration is a Laserfiche component that allows users to take advantage of Laserfiche features when working with Microsoft Word documents, Excel spreadsheets, PowerPoint presentations, and Outlook messages in their native Microsoft Office applications. Office Integration allows you to save Office files directly to your repository from the Office ribbon or toolbar, and to make changes to a document's metadata without leaving Office. For more information, see Saving to Laserfiche from Microsoft Office, Sending Emails to Laserfiche from Microsoft Office, Working with Microsoft Office Documents, and Attaching a Laserfiche Document in Outlook.

Repository Administration

Repository Administration allows you to configure settings for your repository, set security, create user-defined metadata types, and manage repository activity. You can use the links at the left to select the repository settings you want to configure. The actual settings will open in the center of the screen.

For more information, see Security, Metadata, and Records Management.

Account Administration

Account Administration allows you to view billing information and configure user accounts for authentication. Account Administration is only available to administrators. For more information, see Getting Started with Laserfiche Account Administration.

Audit Reports

If you have chosen to audit actions in your repository, you can use Audit Reports to view that information. You can create reports to analyze audit data, view the information as a chart, filter it to include only the information relevant to you, and export the data for use in spreadsheet programs such as Microsoft Excel. If you are in another section of your repository, you can quickly switch to Audit Reports by clicking your user name in the top right corner and selecting Audit Reports. For more information, see Auditing.

Laserfiche Mobile

While Laserfiche is available directly from your phone's web browser, you can also install the Laserfiche Mobile app for iOS or Android. Laserfiche Mobile includes much of the same functionality available in the browser version of Laserfiche, and also allows you to upload documents directly from your camera's phone or photo gallery, along with better support for printing and exporting from your phone. For more information, see Accessing Laserfiche on Mobile Devices.

Public Portal

Laserfiche Public Portal is a user-friendly public portal site for providing read-only access to documents in your repository. You can specify exactly which documents to make publicly available, and customize the appearance and behavior of the site to best suit your organization's needs. For more information, see Public Portal.

Locally Installed Applications

Some components of Laserfiche will be installed directly on your computer, and can be accessed from your computer's desktop.

Laserfiche Snapshot

Laserfiche Snapshot creates pages and text files from electronic files (e.g., a Word document, a Web page, a text editor, etc.) and to store the resulting documents in Laserfiche. This eliminates the need for printing electronic files to paper, then scanning them into Laserfiche. For more information, see Capturing Documents with Laserfiche Snapshot.

Laserfiche Connector

Laserfiche Connector provides a streamlined experience for integrating Laserfiche with line of business applications such as Customer Relationship Management (CRM) and Enterprise Resource Planning (ERP) systems, without needing to write code. Laserfiche Connector integrates easily through user-defined hotkeys and embedded icons. For more information, see Creating Documents with Laserfiche Connector or Searching with Laserfiche Connector.

Laserfiche Import Agent

Import Agent is a tool for automatically retrieving files stored in a Windows folder and importing them into a Laserfiche repository. During the import process, Import Agent can process the files (e.g., perform OCR), add metadata (e.g., populate fields, add tags), and perform additional tasks. For more information, see Using Import Agent.

Laserfiche Quick Fields

Laserfiche Quick Fields is an automated data capture solution that allows you to scan or import and process documents using a wide variety of tools. For more information, see Welcome to Laserfiche Quick Fields.

Laserfiche Client

The Laserfiche Client is a locally-installed way of accessing your Laserfiche documents and folders. The Laserfiche Client is primarily intended for bulk import and export of documents. For more information, see the Laserfiche User Guide.
