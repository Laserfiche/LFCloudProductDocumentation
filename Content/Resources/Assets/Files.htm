Files

Files lets you import various types of files directly into process automation. Unlike a repository, all workflow designers can use the files imported here. Once imported, you can use the files in the Attach Electronic Document and Update Word Document activities.

Importing a File

  1. Click New in the top left of Files.
  2. In the dialog, click Choose File to select the file you want to import. The File Name textbox populates with the name of the file.
  3. Type the name you want to use to refer to the file in your workflow in the Asset Name text box.
  4. Optional: Type a description of the file content.
  5. Click Finish.

The new file appears in the list.

Note: You can import up to file attachments. Learn more about service limits

Modifying a File

  1. Double-click the file you want to modify in the list in Files.
  2. In the dialog, make any modifications to the file. When you modify the name of file, the new name is updated and used throughout Workflow.
  3. Click Save.

Exporting a File

  1. Select the files you want to export.
  2. Click Export above the list in the toolbar.
  3. Type a name for the exported file.
  4. Click Export.
