Configuring Process Notifications

You can configure a process to send email notifications for when a process instance terminates or when a process instance is suspended.

To configure process notification email messages

  1. Open the process diagram for the process of interest.
  2. Click on the Process Properties icon in the top right.
  3. Under the Process notification section, select when you want email notifications sent out.
  4. Use the Edit button, to configure the contents for each type of notification. In addition, you have the option of attaching a copy of the instance error log with the email.
    • Click Show Reply to/CC/BCC to set additional features. Reply to is empty by default. Click Hide Reply to/CC/BCC to hide these options.
    • You may use variables in each of the address fields and in the subject, attachment, and body fields of the message
