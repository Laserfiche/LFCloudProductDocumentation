Creating Custom Reports

To Create a New Report:

  1. Click New on the Manage page to open the Create Report dialog.
    1. Provide a report name and optional description.
    2. Select the process to report on from the drop-down menu.
    3. Click Create to open the Report Designer.

The Custom Report Designer

The Designer window provides an interface to create and view data collected by your processes.

Designer Navigation

The Toolbar Actions and Chart Types page explains the top toolbar and it's elements.

The left pane contains a list of the available fields for use in the report, and includes both form fields from the selected process and system variables. Search for specific fields using the Search fields box at the top of the list.

The main panel contains the field assignment areas for setting which values are used and how they are grouped, settings for filtering and sorting the data and setting up drilldowns, and the report display area.

Adding Data to the Report

Any field or variable available in the left pane under the process name and System Variables can be used as a value in the report designer. The items can be drag-and-dropped into the Value field in the center pane to get started, and can also be dragged into the Groups field for further refinement.

Note: Forms variables containing collections or tables are not currently supported. To report on processes that contain collections or tables, use the Business Process monitor page to download all instances and use a spreadsheet application for reporting. Alternatively, you could create a forms report to view the submission data using the Business Process report page.
