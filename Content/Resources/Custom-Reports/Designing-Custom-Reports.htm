Designing Custom Reports

To Create a New Report:

  1. Click New on the Manage page to open the Create Report dialog.
    1. Provide a report name and optional description.
    2. Select the process to report on.
    3. Click Create to open the Report Designer.

The Custom Report Designer

The Designer window provides an interface to create and view data collected by your processes.

Show Me:

Designer Navigation

The Toolbar Actions and Chart Types page explains the top toolbar and it's elements.

The left pane contains a list of the available fields for use in the report, and includes both form fields and system variables. Search for specific fields using the Search fields box at the top of the list.

The main panel contains the field assignment areas for setting which values are used and how they are grouped, settings for filtering and sorting the data and setting up drilldowns, and the report display area.
