Integrations

Integrations provide a common location for creating and maintaining connection profiles to 3rd party applications and data.

The integrations available are:

  • Remote Agents: Connect to secured resources through an agent
  • Data Sources: Connect to external data from common database providers
  • Application Connections: Connect to software from Salesforce, Redtail, and Microsoft Dynamics 365
  • Web Service Connections: Connect to external web services.

To create a new connection profile, select the type of connection to be created, then click New and complete the wizard. You can also click the arrow next to New and select the type of connection directly.
