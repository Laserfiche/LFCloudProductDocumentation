Actions Using GraphQL

Prerequisite:

You are required to have GraphQL support to run all actions starting with [GraphQL]. If you see the error message "The following resource is not authorized: <resource>" after attempting to run a GraphQL action, please contact Ellucian to setup GraphQL and add the resource for you. Resources within Ellucian that require GraphQL access are: section-registrations, academic-levels, grade-schemes, sections, courses, academic-periods, course-statuses, section-registration-statuses, section-instructors, and persons.

  1. [GraphQL]Search Section Registrations by Person ID
  2. [GraphQL]Search Section Registrations by Person ID and Term ID
  3. [GraphQL]Search Sections by Course ID and Term ID
  4. [GraphQL]Search Courses by Subject ID
  5. [GraphQL]Search Courses by Subject ID and Academic Level ID
