Add Academic Discipline Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used to add a discipline to a students academic program.

  • Category: Student Academic Program
  • Supported in: Workflow

Input Fields

Field Description

Student Academic Program ID*

The Guid identifier of the Student Academic Program.

This field can be retrieved from the Search Student Academic Programs by Person ID action.

Academic Discipline ID*

The Guid identifier of the Academic Discipline to be added.

This field can be retrieved from the Get Discipline List action.

Output

This action does not have any output tokens.

Notes:

If an Academic Discipline ID that already exist in the current student academic program is entered, the duplicate ID will be filtered out by our application. Duplicate Academic Disciplines will not be added in Ellucian.

If the enrollment status of the current student academic program is not active, Academic Disciplines can not be added.
