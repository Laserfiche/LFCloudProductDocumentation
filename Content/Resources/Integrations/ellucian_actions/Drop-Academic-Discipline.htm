Drop Academic Discipline

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used to drop a discipline from a students academic program.

  • Category: Student Academic Program
  • Supported in: Workflow

Input Fields

Field Description

Student Academic Program ID*

The Guid identifier of the Student Academic Program.

This field can be retrieved from the Search Student Academic Programs by Person ID action.

Academic Discipline ID*

The Guid identifier of the Academic Discipline to be dropped.

This field can be retrieved from the Search Student Academic Programs by Person ID action.

Output

This action does not have any output tokens.

Notes:

If an Academic Discipline ID that is not in the current student academic program is entered, an error will be thrown indicating that the Academic Discipline ID is not found.

You cannot drop an Academic Discipline if the enrollment status of the current student academic program is not active.

You cannot drop the last major in the student academic program. Ellucian will throw an error if this is attempted.

If the error "the transaction timeout expired; nested exception is org.hibernate.TransactionException: transaction timeout expired" occurs, try again or contact Ellucian support for help.
