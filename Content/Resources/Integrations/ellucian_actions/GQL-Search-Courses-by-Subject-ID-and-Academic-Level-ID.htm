[GraphQL]Search Courses by Subject ID and Academic Level ID Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used for searching all the courses for a given Subject.

  • Category: Course
  • Supported in: Workflow and Lookup

Input Fields

The input fields are the same as the Search Courses by Subject ID and Academic Level ID action.

Output Fields

This action may return multiple results.

The output fields are the same as the Search Courses by Subject ID and Academic Level ID action.

Note:

The results will be sorted alphabetically by the "Title" field (ascending order).

The Academic Level output field will always be the same as specified as the input field.
