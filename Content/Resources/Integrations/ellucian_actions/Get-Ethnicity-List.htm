Get Ethnicity List Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action returns a list of objects of which each is an ethnicity.

  • Category: Other
  • Supported in: Workflow and Lookup

Input Fields

This action does not have any input fields.

Output Fields

This action may return multiple results.

Field Description

Ethnicity ID (Value)

A unique identifier Guid for this ethnic group, used for updating an ethnicity (ideally used as the value for a drop down).

Title

The Ellucian display name of the ethnic group (ideally used as the label for a drop down)

Note: The results will be sorted alphabetically by the "Title" field (ascending order).
