Get Marital Status List Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action returns a list of objects of which each is a marital status.

  • Category: Other
  • Supported in: Workflow and Lookup

Input Fields

This action does not have any input fields.

Output Fields

This action may return multiple results.

Field Description

Code

A short form of the status.

Marital Category (Value)

The value used to update the status (ideally used as the value for a drop down).

Marital Status ID

A unique identifier for this status.

Title

The Ellucian display name of the status (ideally used as the label for a drop down)

Note: The results will be sorted alphabetically by the "Title" field (ascending order).
