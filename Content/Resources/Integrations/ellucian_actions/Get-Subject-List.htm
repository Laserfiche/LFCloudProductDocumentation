Get Subject List Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action returns a list of objects of which each is a Subject.

This action can be used as input for actions that require subject information.

  • Category: Other
  • Supported in: Workflow and Lookup

Input Fields

This action does not have any input fields.

Output Fields

This action may return multiple results.

Field Description

Subject ID (Value)

The Guid identifier of the subject.

The value used by update actions (ideally used as the value for a drop down).

Title

The full name of a subject (for example, 'Mathematics', and ideally used as the label for a drop down).

Abbreviation

The abbreviated name of a subject (for example, 'Math').

Note: The results will be sorted alphabetically by the "Title" field (ascending order).
