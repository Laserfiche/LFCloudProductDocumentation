Register Section Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action could be used for registering a course section.

  • Category: Section Registration
  • Supported in: Workflow

Input Fields

Field Description

Section ID*

The Guid identifier of the section the student is attempting to register for.

Retrieve this from one of the Search actions for "Section"

Person ID*

The Guid identifier for the Student that is attempting to register for a section.

Retrieve this from one of the Search actions from the "Student" model, as the field "Person Id".

The student must be valid for enrollment in the academic period that the section is from.

Academic Level ID*

The Guid identifier for the Academic level of the section the student is attempting to register for.

Retrieve this from one of the Search actions for "Section" or from the Get Academic Level List action.

The value must be from the list of valid Academic levels for the section.

Credit

Consists of 1 sub-field: registrationCredit.

registrationCredit is an integer representing how much credit the student will receive.

Valid values are based on the minimum, maximum and increment fields from the "CourseCredit" fields gotten from the Search/get actions for "Section" or "Course".

Optional, a default of the minimum value for the section/course credit will be selected if not specified.

Grade Scheme ID

The Guid identifier for the Grade Scheme of the section the student is attempting to register for.

Retrieve this from one of the Search actions for "Section".

The value must be from the list of valid Grade Schemes for the section.

If not using the default value, the academic level must be a part of the acceptable academic levels for that grade scheme. If not, a third-party error will be encountered when trying to run the action.

Optional, the value will default to the first Grade Scheme if not specified.

Output Fields

This action returns a single result.

Field Description

Section Registration ID

The Guid identifier of the registration created/updated by this action.

Note: If an error occurs, this value will not be returned.

Notes:

If the student was previously enrolled in or is currently enrolled in the section, this action will act as an update action. It will ensure the registration is set to "registered" and will try to update the other fields in the case that they are different than what is currently in the existing registration. (If the previous registration is "not registered", after executing the register section action, the credit field will not be updated due to Ellucian update api behavior and will be reverted back to old value.)

The student must be enrolled in the academic period and be eligible to enroll in the course.
