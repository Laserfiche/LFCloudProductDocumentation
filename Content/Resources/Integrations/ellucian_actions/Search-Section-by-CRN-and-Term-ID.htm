Search Section by CRN and Term ID Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used to find section information.

  • Category: Section
  • Supported in: Workflow and Lookup

Input Fields

Field Description

Subject ID*

The Guid identifier of the subject to filter the courses by.

Use the Get Subject List action to find the Subject ID.

Academic Level ID*

The Guid identifier of the academic level.

Use the Get Academic Level List action to find the Academic Level ID.

Output Fields

This action may return multiple results.

The output fields are the same as the Search Courses by Subject ID action.

Note: The results will be sorted alphabetically by the "Title" field (ascending order).
