Search Sections by Course ID and Term ID Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used to find section information.

  • Category: Section
  • Supported in: Workflow and Lookup

Input Fields

Field Description

Course ID*

The Guid identifier of the course that the section belongs to.

Term ID*

The Guid identifier of the reporting academic period with the section.

Use the Get Term List action to find the Term ID.

Output Fields

This action may return multiple results.

The output fields are the same as the Search Section by CRN and Term ID action.

Note: The results will be sorted alphabetically by CRN ascending order.
