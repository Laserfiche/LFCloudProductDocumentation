Search Student by Banner Username Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used for searching all of the personal information for a student and returns one or no results.

  • Category: Student
  • Supported in: Workflow and Lookup

Input Fields

Field Description

Banner Username*

Is the Username for the searched student.

User must input Banner Username exactly as is on Ellucian.

Output Fields

Returns a single result set.

The output fields are the same as the Search Student by Banner ID action.
