Search Students by Email Action

A field marked with an asterisk "*" indicates the field is a required input for this action.

This action can be used for searching all of the personal information for a student and returns one or no results.

  • Category: Student
  • Supported in: Workflow and Lookup

Input Fields

Field Description

Email*

Is one of the emails for the searched student.

User must input Email exactly as is on Ellucian.

Output Fields

Returns a single result set.

The output fields are the same as the Search Student by Banner ID action.

Notes:

The "Email/Email Address" type that is mapped in the connection will be used for the search. So "Email/Email Address" is mapped to "School(school) Email Address" then only Students with the inputted "School" email will return in the results. A person with the inputted Email but of a different type (e.g. "Family (family) Email Address") will not be included in the results.

The results will be sorted alphabetically by the student's last name, followed by the student's first name.
