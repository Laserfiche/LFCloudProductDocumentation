Installing Laserfiche Bots

This section contains the information you'll need to successfully install the Laserfiche bot recorder and local tool. For information about installing bot workers, see the worker help documentation.

System requirements

Supported Operating Systems:

  • Windows 10
  • Windows 8.1
  • Windows Server 2019
  • Windows Server 2016
  • Windows Server 2012
  • Windows Server 2012 R2

Software requirements

  • Microsoft Visual C++ 2017 Redistributable (x64) -14.11.25325
  • Microsoft Visual C++ 2017 Redistributable (x86) -14.11.25325
  • .NET 4.7.2

Installing Laserfiche Bot Recorder and Local Tool

You can install the bot recorder and local tool from the following places:

  • When adding a recording in the bot designer. Drag Add a Recording from the Toolbox to the canvas and click Open bot recorder.
  • From the Test bot button in the bot designer.
  • From the Run button in the bot Manage page, when you select to run the bot locally.
  • From the Dedicated Bot Remote Agent's Worker tab.

Installation

  1. Download the autorun package.
  2. Click the .exe file to open it.
  3. Click Local Tools/ Recorder Installation.
  4. In the Welcome step, click Next.
  5. In the End-User License Agreement step, read the license agreement and select I accept the terms in the License Agreement if you agree to the terms of the license agreement. Click Next.
  6. In the Custom Setup step, ensure Recorder and Local Tools are selected.
  7. Specify the desired install directory on this page. Click the Disk Usage button to see how much space is available on the specified disk and how much space the installation requires, to ensure you have enough space to install the product. Click Next.
  8. On the Laserfiche Service Settings step, provide a Windows account for the Laserfiche Robotic Process Automation Agent service to run as.
  9. In the Ready to Install step, click Install to start the installation process.
  10. The Prerequisites step may appear if Laserfiche Bots needs to install additional components. A green check will appear next to installed items and a red X will appear next to items that have not been installed. If one or more of the items on the list has a red X next to it, install that item and then click Retry. Click Next.
    11. Important: Bypassing prerequisites may cause bots to function incorrectly and is not recommended.
  11. When the installation is complete, click Next.
  12. We recommend leaving the Open the Laserfiche Bots Local Tool checkbox selected. This will start the local tool (which provides an action history as bots run). It will also add the Laserfiche Bots icon to your tray menu.
  13. Click Finish to exit the installation wizard.

Configuring Bots to Run with a Remote Agent

Remote agents perform work outside Laserfiche Cloud's security domain and are necessary if you want to run bots via a timer starting event or the Invoke Bots activity in Workflow. You can also run bots via remote agent from the Manage page.

Bots will automatically be assigned to run with the Dedicated Bot Remote Agent that has been created for you. Learn more.

Note: If you configured your own remote agents to run bots in previous versions of Laserfiche Cloud, you will need to migrate your bots to the new Dedicated Bot Remote Agent. Learn more.

It's important to note:

  • Although the remote agent is created for you, you must still install workers.
  • Additionally, the Laserfiche Bots local tool must be running for associated bots to run on the worker. You can launch the local tool from the Windows Start menu after Laserfiche Bots has been installed.

Accessing Laserfiche Bots

  1. Navigate to the Process Automation Overview page in Laserfiche Cloud.
  2. Click Bots.
