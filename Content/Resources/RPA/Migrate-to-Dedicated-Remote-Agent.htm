Migrate to Dedicated Remote Agent

Bots can only run on the Dedicated Bot Remote Agent, a pre-configured remote agent that is specific to Laserfiche Bots. You must install workers for the new dedicated remote agent before it's operational.

Installing workers for the dedicated remote agent

  1. Navigate to Remote Agents from the Integration page in Process Automation.
  2. Select the Dedicated Bot Remote Agent.
  3. Select the Workers tab and follow the instructions install and register a worker. Learn more.

Migrating to the dedicated remote agent

  1. From the Manage page, select a bot that is using a previously created remote agent. When selected, these bots will display a red error icon in the right pane.
  2. If you haven't installed a worker for the dedicated bot remote agent, follow the instructions above to install one now.
  3. Navigate to the Remote Agents tab, and click Migrate.
