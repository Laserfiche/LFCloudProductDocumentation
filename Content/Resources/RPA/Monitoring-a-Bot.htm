Monitoring a Bot

The Monitor page, Action History, and Manage page all provide details about individual instances of a bot.

Instance details

On the Monitor page, you can search for and view instance information such as start time, duration, and initiator.

Note: Full details are only available on the Monitor page for bots run via a remote agent. Bots run on your local machine or from the Test option in the bot designer will not be listed on this page.

  1. Navigate to the Monitor page in Laserfiche Cloud by using the app picker to select Process Automation, and then Bots. Then, click Monitor at the top of the page.
  2. Use the options in the left column to search for an instance.
    • Use the Quick Search drop-down menu to find bots that ran recently.
    • Use the other search criteria to search based on bot name, keywords in the instance, current status, and more. Expand the Start Information section to search based on when the bot started.
  3. After defining your criteria, click Search. Click Reset to clear your search criteria.
  4. Click a column title to sort instances based on the values in that column.
  5. Click an instance name to see the instance's history. The history details each action the bot took and the status of the action.
  6. Click the Input and Output tab, to see the input and output parameters for the instance.

Step-by-step action history

In the Action History, you can see a step-by-step log of each action the bot took.

When you run a bot locally (by testing the bot from the designer), the Action History will open in the bottom-right of your desktop, providing information on how the bot is running and its current status.

Tip: Click the arrows in the top right of the Action History to see an expanded view.

  • If your bot includes a wait activity, the Action History will prompt you to click Mark as complete to advance the bot.
  • You can end running bots from the Action History by clicking the Stop bot button.
  • Click the Token and Input/Output Parameter tabs to view the tokens and parameters used in the bot instance. If a token or parameter has multiple values, click the View values link to see a list of all values.
  • Use the View previous link in the top left to view previously run bot instances. You can return to the current instance by clicking the View next link in the top right.
  • If you accidentally close the Action History, you can re-open it by clicking the Laserfiche Bots icon in the system tray section of the taskbar.

Statistics

For bots that run via remote agent, you can see statistics for average bot duration, number of running instances, and more in the Instances tab on the Manage page.

Ending a running bot

You can end running bots from the Action History by clicking the Stop bot button.
