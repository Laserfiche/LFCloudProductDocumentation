Running a Bot

Before running your bot, take note of the following:

  • If you are running bots from the bot designer or locally from the Manage page, only one bot can run at a time. If a bot is suspended at a wait activity, you must wait for the bot to continue before starting the next bot. If you run multiple bots at the same time via remote agent, the bots will be queued and will run in turn.
  • If you need to run your bot in administrator mode (e.g., if your bot is installing software, updating drivers, or changing system settings), in the bot designer, select the Settings button in the toolbar and select Run as administrator.
  • Do not touch the keyboard or mouse when a bot is running, otherwise, the bot may lose focus and stop running.

Running a bot via remote agent

Bots configured to run on a remote agent can be started in the following ways:

  • From starting event that runs the bot based on a timer
  • From a workflow
  • From the Manage page.

Before running a bot via remote agent ensure the following:

  • The Dedicated Bot Remote Agent has workers installed.
  • If the bot was created with a previous version of Laserfiche Bots, you have migrated your bot to the Dedicated Bot Remote Agent. Learn more.
  • The latest version of bot local tool is installed and running on the machine the bot will run in. You can launch the tool from the Windows start menu.
  • If the bot needs to run in administrator mode, ensure the local tool is running in administrator mode as well.
  • A user is signed into the machine the bot will run on.

Running a bot locally

All published bots can be run from the Manage page.

  1. Select the bot you want to run.
  2. Click the play button .
  3. Select Locally from the drop-down menu.
  4. Click Open Laserfiche Bots WebLauncher.
    • If you specified input parameters for the bot, you'll be prompted to enter values for those parameters.

Tracking bots

The Monitor page, Action History, and Manage page all provide details about individual instances of a bot. Learn more about monitoring a bot.
