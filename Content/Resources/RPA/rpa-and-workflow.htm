How Do Laserfiche Bots Fit into Laserfiche Process Automation and Workflow?

Robotic Process Automation fits natively into the larger Laserfiche Cloud Process Automation platform, and the bot designer is available from the Laserfiche Cloud Process Automation homepage.

Learn how business processes, workflows, and bots work together.

Run Bots via Remote Agents

Laserfiche bots make use of remote agents. Remote agents provide a secure connection that allows bots to run remotely on other machines and then send results back to process automation.

Invoke Bots from Workflow

Bots are accessible from within workflows through the Invoke Bot activity. You can use workflow to pass in tokens for a bot to use (as input parameters) while performing its recorded tasks. Additionally, bots can produce output parameters that can be passed back to workflow and used in other activities in the workflow.
