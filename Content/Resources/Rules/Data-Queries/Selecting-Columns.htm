Selecting Columns

The SELECT section of the query editor lets you select the columns from the table that you want to query.

To specify columns:

  1. Click Add.
  2. Select the name of the column you want to query under Column.
  3. Select the number of unique entries you want to return in Quantifier.

    4. Example: If $1 is repeated twice, use DISTINCT to count $1 once and ALL to count each appearance of $1.

  4. Optional: Under Function, select an aggregate function to apply to the data.

    5. Example: Using the Max operator, you can find the maximum salary for all employees in an Employee table.

  5. Type a name for the output parameter containing the result of the query under Output Parameter. You can use the output parameter name to reference the configured statement in the Run Rule Workflow activity.
