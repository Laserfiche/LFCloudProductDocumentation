Ordering the Output Priority

The output priority ordering is first determined by the order of the output fields from top to bottom or from left to right, and second, by the order of the values in the predefined list for each output field.

Output priority ordering matters for configuring and using an output order hit policy or priority hit policy.

To configure output priority on the decision table:

  1. Hover over an output field.
  2. Select the grip .
  3. Drag and drop each output field to specify the order of importance of the outputs.
  4. Reorder the output entries as follows.
    1. Click on the output's type to bring up the output's properties pane.
    2. Drag and drop the output entries in the predefined list into a different order.
