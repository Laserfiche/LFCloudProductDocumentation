Email Display Names

Create sender display name for email messages sent as part of a business process. By default, business process email notifications sent by an email service task or a user task email notification are shown as being sent by Laserfiche Notification. Create additional display names that process designers can select when configuring the email service task or user task.

Creating and Modifying an Email Display Name

You must be a Process Administrator, Process Manager, or Process Asset Administrator to add or edit email display names.

  1. Open the Settings page.
  2. Select the Email Display Names tab and click New.
  3. Type the desired display name and note the preview for how the display name will appear.
  4. Click Create.

Note: You can create up to email display names. Learn more about service limits

To modify an email display name:

  1. Open the Settings page.
  2. Select the Email Display Names tab.
  3. Select the email display name in the list.
  4. Click Edit () in the top right.
  5. Modify the display name as desired and note the preview for the new display name.
  6. Click Update to complete the change.
