Creating a Timer Event

Use the Set Timer step to specify a start time and recurrence schedule.

  1. Use the blue text and drop-down next to Start On to select a date, time, and time zone when the event starts. Note that the time zone you select here is your time zone. For example, if you are on the east coast and set a starting event to start at 9 AM Eastern Standard Time (EST) by selecting "Atlantic Time (Canada)", the starting event will start at 9 AM EST.
  2. Select how often you want the event to repeat next to Repeats. (More information below)
  3. Set up how often you want the workflow or bot to repeat after the schedule starts when you select Rerun starting event every. For example, when you set a daily schedule and rerun the starting event every 15 minutes for one hour, the starting event runs five times a day.

    4. Note: The Rerun starting event every condition doesn't cause the workflow or bot to repeat beyond the end of the day. For example, if you schedule a workflow to run every 15 minutes for one hour and set its starting time to be 11:30pm daily, the workflow will run only twice daily: at 11:45pm and 11:30pm. It will not run at 12:00am, 12:15am, etc. because those times are part of the next day. On the next day, it will start at 11:30pm again.

    Important: As a best practice, timer repetitions should be at least 15 minutes apart.

  4. Specify when you want the event to stop running when you select Stop the timer on.

Note: You can create up to starting events. Learn more about service limits

Testing a Timer Event

Click Show Tests to view the dates and times when the schedule will run. The Test count textbox lets you specify the number of dates and times you want to see from the schedule. For example, when you type "5" in the textbox, the first five dates and times when the starting event will run will appear.

Setting Up a Repeat Schedule

You can use the following types of repeat options:

Repeat Options
NameDescription
NeverStart the workflow or bot once.
DailyStart the workflow or bot everyday, weekday, or number of days (e.g., every 1 day repeats everyday, every 2 days repeats every other day).
WeeklyStart the workflow or bot on the selected days or number of weeks (e.g., every 1 week repeats every week, 2 every other week).
MonthlyStart the workflow or bot every month on a specific day (e.g., on the first Sunday only repeats the first Sunday of each month).
