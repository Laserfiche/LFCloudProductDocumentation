Projects

Projects let you organize access to resources and limit access to data collected by those resources.

Overview of Project Security

By grouping your business processes, rules, lookup tables, and other process automation resources into projects, you can control who can edit those resources and who can see the data collected by those resources.

The Global Project

The global project is the default project that is accessible to all Laserfiche account users that have the Access to Process Automation security role assigned. Any resources created within this project may be accessed by any other project.
