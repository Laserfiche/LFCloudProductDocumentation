Add Text Box

This activity adds a text box to a page of a Laserfiche document. A text box is a small, moveable box that contains text. Learn about tokens this activity produces

To set up this activity:

  4. In the Text Box Options property box, configure settings for the text box using the following options:
    • Text: Type the text that will appear inside the text box. Click the Token button (right arrow) to add a token.
    • Font Size: Select how large you want the text to be.
    • Fill Color: Select the color inside the text box.
    • Opacity (%): Select the amount of visibility of the text box. At 100% opacity, the text box is completely visible. At 0%, it is transparent.
    • Rotation: Select how many degrees you want to rotate the text box when it is added to the page: 0, 90, 180, or 270 degrees.
    • Color: Select the border color of the text box. The default color is black.
    • Thickness (px): Type how thick, in pixels, the text box's border will be, where 0 pixels will not have a border.
    • Style: Select the border style of the text box.
    • Page: Select the page on which you want to place the text box. Click the Token button (right arrow) to add a token.
    • Location: Select where the text box will appear on the Laserfiche document. Select Custom to place it on X and Y coordinates (in pixels) starting from Top/Left.
    • If you have more than one text box in the same location, use Merging to select how you want to display the text box.
      • None: Places the new text box on top of the existing text box.
      • Replace: Replaces the existing text box with the new text box.
      • Append: Adds the text of the new text box to the existing text box.
      • Cascade: Places the new text box slightly offset from the existing text box. Show me what this looks like

  5. In the Text Box Protection property box, specify the rights for who can view and modify the text box.
    • Public: Anyone who has access to the document can view and modify public annotations.
    • Protected: Anyone who has access to the document can view protected annotations, but only the user who created the annotations can modify them.
    • Private: Only the user who created the private annotations can view and modify the annotations.
  6. In the Add Text Box To property box, select the Laserfiche document you want to place the text box on. Learn more about selecting an entry
