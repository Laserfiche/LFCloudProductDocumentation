Assign Field Values

This activity assigns and reassigns a template and field values in a Laserfiche entry. A field value is a piece of information in a field about a document or folder. This could be, for example, an author’s name, an invoice number, or an address. A template is a collection of fields. Only one template may be applied to a document or folder at a time.   5. When you unassign or assign a template, select Remove fields from unassigned templates to remove all fields associated with the entry's starting template. Clear the checkbox so that the template's fields remain associated with the entry even though the template is removed.
  6. In the Field Values property box, click Choose fields... to add fields.
  7. Select the fields from the list you want to assign values to. If a template is assigned in the Template property box, only the fields in that property box are available.
  8. Click OK.
  9. Configure the field in the property box.
