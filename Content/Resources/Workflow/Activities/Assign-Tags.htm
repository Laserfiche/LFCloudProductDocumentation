Assign Tags

This activity adds and removes tags from an entry. Tags are metadata in Laserfiche that provide a way to categorize and search for documents. You can use tags to categorize the type of information in the entry and then later, search for documents categorized by that type of information.

To set up this activity:

  4. In the Assign/Remove Tags property box, click Configure tags... to select the tags you want to assign or remove.
  5. Select the tags you want to assign in the column under Assign and the ones you want to remove under Remove.
  6. Optional: When you add or remove a tag, add a comment to the tag under Comment to remember why you are assigning or removing the tag.
  7. Click OK.
  8. In the Assign Tags To property box, select the Laserfiche entry you want this activity to act on. Learn more about selecting an entry
