Delete Entry

This activity deletes an entry in your Laserfiche repository. An entry can be a document or folder.

To set up this activity:

  4. In the Options property box, select additional deleting options.
    • Purge entry after deleting: Permanently deletes the entry from the Laserfiche repository, purging the entry from the Laserfiche recycle bin after the entry has been deleted. Note that purging an entry is irreversible.
    • Only delete empty folders: Only deletes folders that contain zero entries.
  5. In the Entry to Delete property box, select the entry you want to delete. Learn more about selecting an entry
