For Each Entry

This activity runs the contained activities on each entry in a collection of entries. Learn about tokens this activity produces

To set up this activity:

  4. In the Entries property box, select the collection of entries you want this activity to operate on by selecting the Edit button , hovering over the activity you want to select the entry from, and selecting Output Entries.
  5. In the Record Entries property box, select List the entries in the workflow's instance details if you want to list the entries in the instance report.
  6. Drag activities from the toolbox pane and drop them into this activity. These are the activities this activity will run.
