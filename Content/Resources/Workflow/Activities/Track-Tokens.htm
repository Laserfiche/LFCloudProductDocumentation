Track Tokens

This activity records the values of specified tokens when this activity runs. You can view these values in the Tokens tab of the instance report. Add this activity to multiple points in the workflow to record how the tokens change throughout the workflow. After you test the workflow, remove the Track Tokens activities from the workflow.

Note: This activity tracks the first 3700 characters of a token's value.

To set up this activity:

  3. In the Tokens to Track property box, specify the tokens.
    • All tokens: Records the values of all the tokens in the workflow at the point when the Track Token activity is run.
    • Specific tokens: Select specific tokens to track within their token category. To select specific tokens:.
      1. Click Select... to choose individual tokens or Select all to choose all tokens in the category.
      2. In the Select Tokens dialog box, select the tokens.
      3. Click Save. The tokens appear in the property box.
