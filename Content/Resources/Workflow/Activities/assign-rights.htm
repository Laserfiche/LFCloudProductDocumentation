Assign Rights

This activity assigns or removes Entry Access Rights associated with an entry for a specified user. If a specific right is not specified, it will not be modified.

To set up this activity:

  4. Click Configure rights... to open the Rights Options dialog.
  5. Select a trustee or specify a token that contains a user ID.
  6. Set the desired options for the entry and click OK.
  7. Select which entry the changes will be applied to. Learn more about selecting an entry

Available Rights Options:

  • Remove all rights: Removes all rights settings for the user from the entry.
  • Assign the following rights: Assigns either the Allow or Deny right for each right type. See the List of Access Rights for a complete list and description of each type.
    • Selecting Inherit rights from parent
    • Select the scope for the activity using the If the entry is a folder, apply the rights to option. See the List of Scopes table for details.
