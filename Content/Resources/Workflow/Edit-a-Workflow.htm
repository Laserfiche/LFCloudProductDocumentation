Edit a Workflow

To edit a workflow, open the workflow using one of the following methods:

  • From the Manage page, either double-click the workflow in the list, or select the workflow and click the Edit button.
  • From the Monitor page, open a workflow instance, click the Details tab, and click view next to the workflow name in the Workflow Details section.
  • From the Design page, click Open a workflow or click the + icon. Select the workflow to edit, and click Open.

Once the workflow is open in the Designer canvas, any changes can be made. To activate the changes: save, publish and test the workflow.
