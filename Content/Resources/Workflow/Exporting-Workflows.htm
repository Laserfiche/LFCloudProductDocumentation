Exporting Workflows and Their Components

Laserfiche lets you export workflows and the associated components of the workflows (like the starting events and rules) so that you can share them with others, as well as have a versioned copy of them offline. You can export a workflow as a .wfi file or save a workflow as a .wfx file. The .wfi file contains one or more workflows and the associated components of the workflows. The .wfx file contains one workflow. You also have the option to export a workflow event report, which is a detailed summary of the actions a user performed in a specific workflow.

To export workflows and their components:

  1. Select one or more workflows in the list on the Manage page.
  2. Select Export in the toolbar.

  3. Type a name for the file.
  4. Click Export.

To save only the workflow:

  1. Open the workflow designer for a specific workflow.
  2. Click Download in the toolbar.
  3. Click Export WFX.

Note: .wfx files can not be re-opened by the cloud version of workflow at this time.

To export a workflow event report: 

  1. Open the workflow designer for a specific workflow.
  2. Click Download in the toolbar.
  3. Click Export Event Report.

Note: For use in troubleshooting purposes only.
