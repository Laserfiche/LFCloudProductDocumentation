Token for Retrieve Document Text

Tokens
Name Description Sample Syntax
Text The text of the specified document. %(RetrieveDocumentText_Text)

The "RetrieveDocumentText" portion of the syntax changes to match the activity's name.

Note: All non-alphanumeric characters, except underscores, are removed from the name. For example, if you rename the activity "Proposal Contents...", the syntax for the Text token will be "%(ProposalContents_Text)".
