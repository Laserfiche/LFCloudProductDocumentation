Tokens for Set Entry Owner Activity

Tokens
Name Description Sample Syntax
New User Name The account name of the new entry owner. %(SetEntryOwner_New User Name)
Old User Name The account name of the entry's owner before this activity assigned a new owner. %(SetEntryOwner_Old User Name)

The "SetEntryOwner" portion of the syntax changes to match the activity's name.

Note: All non-alphanumeric characters, except underscores, are removed from the name. For example, if you rename the activity "Assign-to-@Initiator", the syntax for the New User Name token will be "%(AssigntoInitiator_New User Name)".
