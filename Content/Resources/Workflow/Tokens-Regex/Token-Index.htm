Token Index

The Apply Index option in the token editor is useful when you want a token to contain only one of the values in a multi-value token or to be able to separate each value in the multi-value token with one or more characters.

The value at index

This options lets you select one of the values in a multi-value token. To select the value, type the value's index number in the text box or click the Token button (right arrow) and add a token. The button below the text box (initially labeled "start") controls the direction in which you are counting. When "start" is displayed, the number 1 in the text box refers to the first item in the multi-value token. When "end" is displayed, the number 1 refers to the final item in the multi-value token.

All values separated by

This option lets you separate each value in a multi-value token with one or more characters, like a comma, semi-color, or space.
