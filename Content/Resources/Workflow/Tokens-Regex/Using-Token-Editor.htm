Using the Token Editor

The token editor lets you temporarily modify a token for the current activity using functions, regular expressions, indexing, and formatting. These editing options are also available in the token dialog. For example, when you apply a function to the Date token in the first of two activities, the function is not applied to the Date token if you use the token again in the second activity.

To modify a token with the token editor:

  1. Click the Token button (right arrow) in the properties pane and add a token.
  2. Click the box with the name of the token in it.
  3. Select any of the options to modify the token within this activity.
    • Apply Function
    • Apply Index
    • Apply Formatting
    • Apply Regular Expressions
  4. Optional: Click Click here to test the token. at the bottom of the editor to test that the token works as expected. If the token is a multi-value token, type your sample values in multiple lines.
  5. Select OK to apply your changes.
