Viewing Usage Statistics

In addition to the instance report feature, you can use the right pane in the Manage page to view a brief overview of the performance of the instances for a selected workflow. Each overview is presented on a card, which reports on the instances in a specific timeframe. The card has the total number of instances started, completed, and terminated, as well as the average duration of all the instances within the specific timeframe and the number of warnings (unexpected events that do not terminate the instance). Opening the Instance Overview

  1. Go to the Manage page.
  2. Select a workflow.
  3. In the right pane, go to the Instances tab. If the pane is closed, click the left arrow button in the top right of the page to open the pane.
  4. Optional: Click Add Card above the card to specify a new timeframe.
