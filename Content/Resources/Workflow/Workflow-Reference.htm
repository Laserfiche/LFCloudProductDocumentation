Workflow Reference

The reference section provides easy access to more detailed information for Workflow Designers. It is broken into categories for Activity, Token, and Regular Expression reference, includes information about the Workflow Templates, and provides an area for more information available outside the documentation set.

 

    Activities
  • A complete list of all available activities, what they do, and directions on their use.
    Tokens
  • Information about tokens and how they are used within a Workflow.
    Regular Expressions
  • Details on the Regular Expression engine within Workflow, and some sample patterns.
    Workflow Templates
  • The Workflow templates include eight fully functional Process Automation workflows that automate common processes.
    Additional Reading
  • Links to articles, whitepapers, and discussion groups for additional resources in the use of Laserfiche Workflow.
