Workflows

Workflows provide no code automation of complex business tasks in Laserfiche using sequences of activities (pre-built, configurable automation elements) in a flow diagram format.

To access workflows, select Process Automation from the app picker, and then click Workflows. Show me what this looks like.

The workflow help pages are organized as follows:

  • Manage: Organize, view, import, and export workflows. Configure workflow options.
  • Design: Create, build, test, and publish workflows.
  • Monitor: View a workflow instance report. A workflow instance is a particular running, terminated, or completed workflow.
  • Reference: In-depth reference material and additional reading topics.

To get started using workflows:

  1. Design a workflow
  2. Manage workflow starting events and notifications
  3. Monitor workflow performance

Note: To learn about the differences between workflow in Laserfiche 11 and Workflow on Laserfiche Cloud, go to Comparing Workflow Versions.
