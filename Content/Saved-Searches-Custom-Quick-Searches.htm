Saving Searches

Once you have configured your search filter (see Search Filters for more information), you can click Save at the bottom left corner of the Search Filters pane. This will save the search so that you can quickly run it again later. In the Save Search dialog box, you can either select Save as a new search and type a name to save it, or select Replace an existing search to select a search you already saved and replace it with the current search. Click OK to save the search.

Once you have saved a search, you can load it by opening the Search Filters pane and clicking Saved Searches. Select a search to load it, and then click Search to run that search.

Most users will only be able to save the saved search for themselves; it will not display to other users. Users with the Manage Repository Configuration privilege can also create saved searches for entire groups, or for everyone.

You can also delete a saved search by selecting it and clicking the Delete button ().

Tip: If you create a date search where the search value is a relative time-frame, such as "in the past 5 days," the search will automatically be updated whenever you run it. This allows you to quickly perform these searches without having to re-configure the date range each time.

Custom Quick Searches

You add additional search types to quick search to better meet your organization's needs. For example, if your organization uses the Created By search often, it can be added to the quick search bar so that you can simply type a user name and search. Any search type that can be configured using search syntax can be created as a custom quick search.

To create or modify a custom quick search

  1. Select Search Filters in the folder browser or Add Filter in the search pane.
  2. At the bottom of the dialog box, select Manage Custom Quick Searches.
  3. Select Add to create a new custom quick search, or select the Edit button () next to the quick search you want to modify.
  4. In the Name option, provide a name for the quick search.
  5. In the Search Query option, specify the search syntax for the search.
  6. In place of the search term, use the %(SearchTerm) token.
  7. In the Apply to option, if you are a user with the Manage Repository Configuration privilege, you can choose whether to apply the quick search to only yourself, one or more specific groups, or everyone. If you do not have this privilege, you can only apply the quick search to yourself.
  8. Click Save.

To delete a custom quick search

  1. Select Search Filters in the folder browser or Add Filter in the search pane.
  2. At the bottom of the dialog box, select Manage Custom Quick Searches.
  3. Select the Delete button () next to the quick search you want to delete. Note that if you shared this custom quick search with other users when you created it, this will remove it from all users.
