Scanning into an Existing Document

In addition to using scanned images to create new documents, you can also scan additional pages into existing documents.

Note: If a document has an electronic file associated with it, such as a Word document or a PDF, you cannot scan into it.

To scan into an existing document

  1. Select or open the document you want to scan into.
  2. If you have selected the document in the Folder Browser, click the Scan button (). If you have opened the document in the Document Viewer, click the document name in the upper left and select Scan.
  3. If you have not yet used Laserfiche Scanning, you will be prompted to install it. See Overview of Laserfiche Scanning for more information.
  4. Select basic or standard scanning mode.
  5. Specify whether you want the newly-scanned pages to be inserted before the first page of the existing document, after the last page, or at a certain point in the document.
  6. Continue scanning as usual. See Basic Mode or Standard Mode for more information.

Note: When scanning into an existing document, you cannot modify the document's name, template, or path.
