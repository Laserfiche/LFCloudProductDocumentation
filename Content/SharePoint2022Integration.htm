SharePoint Integration 2022

The Laserfiche SharePoint Integration allows you to quickly and easily save documents in SharePoint to your Laserfiche repository. Configure profiles to ensure each SharePoint content type is saved in the correct location in Laserfiche with the correct metadata.

Saving to

  1. Select the document you want to save to Laserfiche.
  2. From the More options button, select Save to Laserfiche.
  3. Sign into your Laserfiche account if you're not already signed in.
  4. After the item has been saved, you can click
    • Go Back to return to your SharePoint library.
    • View File to view the document in Laserfiche
    • Close to close the dialog and return to the Laserfiche sign out page

Configuring the Integration

The Laserfiche Administration page lets you specify how SharePoint documents will be saved to Laserfiche. Sign in and select the task you want to perform from the menu on the top of this section.

Profiles

Profiles govern how documents in SharePoint will be saved to Laserfiche. You can create multiple profiles for different SharePoint content types. For example, you may want applications stored differently than invoices, and thus you’ll create separate profiles for each.

Note: Ensure that each profile has a different Laserfiche template.

To create a profile

Click the Create Profile button and fill out the following:

  • Document Name: How the document will be named in Laserfiche. The document will be automatically named with its SharePoint file name.
  • Laserfiche Template: The Laserfiche template that will be applied to the documents. Select a template from the drop-down menu
  • Laserfiche Destination: Where the document will be saved in the Laserfiche repository. Click the Browse button and navigate to the folder in your Laserfiche repository where you want to save the documents.
  • After import: What will happen to the document in SharePoint after its imported into Laserfiche
    • Delete the SharePoint file.
    • Replace SharePoint file with a link to the document in Laserfiche.
    • Leave a copy of the file in SharePoint.
  • Field Values: The SharePoint columns that will map their values to Laserfiche fields from the template selected above. When a document is saved to Laserfiche using this profile, the value in the specified SharePoint column will be copied to the corresponding Laserfiche field.

    • Note: Required Laserfiche fields must be mapped to SharePoint columns.

    • Click Add field to map more fields in the template.

To edit a profile

  • Click the Edit button to edit an existing profile.
  • Click the Remove button to remove a profile.

Profile Mapping

In this tab, you can map a specific SharePoint content type with a corresponding Laserfiche profile. This profile will then be used when saving all documents of the specified SharePoint content type.

  • Click Add Mapping to match a SharePoint content type with a Laserfiche Profile.
  • Click the Edit button to edit an existing mapping.
  • Click the Remove button to remove a mapping.
