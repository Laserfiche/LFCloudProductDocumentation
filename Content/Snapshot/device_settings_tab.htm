Device Settings Tab

The Device Settings tab, in the Laserfiche Snapshot Printing Preferences dialog box, contains settings for paper size, orientation, and graphics resolution, which can be used to customize the images that will be displayed in the Laserfiche repository.

  • Paper size: Set paper size and units. Valid paper sizes are listed in this option and the corresponding page sizes are displayed in the Paper width and Paper height fields. If the desired paper size is not listed, select Edit Paper List to and specify your own width and height. Page sizes can be specified in increments of 0.01 inch, 0.1 mm, or pixel units.
  • Orientation: Select the appropriate page orientation for the document to be printed. If you select Landscape, you can choose Rotate Landscape image to Portrait to rotate the image for the Snapshot process.
  • Create faxable image: Select this option to force Snapshot to create images that are either 204x98 DPI or 204x196 DPI.
  • Graphics resolution: Specify the desired resolution for your images by selecting a preset resolution or entering your own. Selecting a resolution will display the corresponding horizontal and vertical resolutions in the fields below. Resolution can be customized by selecting CUSTOM and modifying the values for horizontal and vertical resolution. The valid range for both horizontal and vertical resolution is 50 - 3000 DPI.
  • Restore Defaults: Allows you to restore the default settings for all printer options.
  • About: Displays copyright and version information for the printer driver.

Note: For applications containing page setup options (e.g., Microsoft Word), it is possible that page orientation, color depth, and other properties selected from the Snapshot Printing Preferences dialog box will be overwritten by the properties set in the Windows application. For these applications, configure the properties from the Windows application.
