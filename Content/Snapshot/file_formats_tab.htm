File Formats Tab

The File Formats tab, of the Laserfiche Snapshot Printing Preference dialog box, displays settings for file format, color depth, and photo quality that will be used when creating images from an electronic file.

  • File format: A list of file formats that can be used to create images. Certain file formats are only available when dealing with a particular color depth (i.e., black and white, grayscale, and color). If you select a TIFF format, you can select TIFF settings for more information.
  • Color depth: Determines if images will be created in black and white (monochrome), grayscale, or color.
  • Photo Quality: Used to improve image quality.
    • Intensity: The brightness of the image. A lower value will produce a darker image, and a higher value will produce a lighter image.
    • Disable photo quality: Disables photo quality image processing.
    • Dithering methods: Floyd-Steingberg, Jarvis-Judice-Ninke, Smooth, Sharp, Stucki and Threshold are the available dithering image filters. The fastest filter is Floyd-Steinberg, because it requires the least computation. Stucki is the slowest filter, with the highest quality and the best output.

Restore Defaults: Allows you to restore the default settings for all printer options.
