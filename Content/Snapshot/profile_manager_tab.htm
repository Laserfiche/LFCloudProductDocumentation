Profile Manager Tab

The Profile Manager tab, in the Laserfiche Snapshot Printing Preferences dialog box, allows you to create, modify, export, import, and delete a printer setting profile. A profile is a collection of printer settings.

  • Select and use an existing profile: If a profile will be used. Selecting a profile will change all printer settings to match the configuration stored in the profile.
  • Add current settings as a new profile: Save the current printer configuration as a profile. Opens the Add new profile dialog box where you can name and describe the profile to be created.
  • Delete an existing profile: Delete an existing profile. Opens the Delete Profile dialog box where you can select the profile to be deleted.
  • Export a profile to a file: Export an existing profile to your computer. Opens the Export Profile dialog box box where you can select the profile to be exported and where it will be saved.
  • Import a profile from a file: Import a profile stored on your computer. Opens the Open dialog box where you can find and select the desired profile.
