Software Requirements for Laserfiche Cloud

Repository

The Laserfiche Cloud repository can be viewed using any of the following web browsers: Microsoft Edge (Chromium), Firefox, Chrome, and Safari for iOS.

  • The Microsoft Office Integration requires the following specifications:
    • Microsoft Office: Microsoft Office 2013, Microsoft Office 2016, and Microsoft Office 2019, Microsoft Office 2021, Microsoft 365 Apps
    • For online viewing and editing of documents using the integration with Microsoft 365, please see Microsoft's documentation on which browsers work with Microsoft 365.
  • Laserfiche Scanning requires the following specifications:
    • CPU: Intel or AMD x64 Processor (2.4 GHz processor)
    • Memory: 1 GB RAM or more
    • Operating system: Microsoft Windows 8.1 and Windows 10

For information on supported browsers for viewing and editing Microsoft Office documents directly within the web client, see Which browsers work with Microsoft 365 for the web and Microsoft 365 Add-ins.

Process Automation

Laserfiche Process Automation can be viewed using any of the following web browsers: Microsoft Edge (Chromium), Firefox, and Chrome for Windows, macOS, and Linux. Process Automation administration and configuration pages may render slowly or not function properly when viewed on mobile devices.
