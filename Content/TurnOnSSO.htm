Turn on Single Sign-On for Users in Laserfiche

  1. In Account Administration, click the Users tab.
  2. Click the name of the user for whom you will be enabling SSO.
  3. In the User Details section, check the check-box for Enable Single Sign-On.
  4. Once Enable Single Sign-On is checked, the Federated ID field will appear. This is set in your identity provider's page.

  5. Click Save to finish.
