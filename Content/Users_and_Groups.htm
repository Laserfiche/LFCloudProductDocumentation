Users and Groups

Users and groups are created and configured in Laserfiche Account Administration, but you can view your users and groups and configure certain options and security settings for them in Repository Administration.

  1. Open repository management by selecting the app picker and then selecting Repository Administration.
  2. On the left, select Users.
  3. View the list of users in the repository in the Users tab, and the list of groups in the Groups tab.
  4. In the General tab, you can review the user or group's name, any repository groups to which they belong, the account type and status, and any security tags that have been assigned to the user or group.
  5. In the Rights tab, you can view the privileges that have been granted to this user or group. See Privileges for more information.
  6. In the Auditing tab, you can view the audit settings for this user or group. See Auditing for more information.

Everyone Group

The Everyone groups is a special group that contains every user in the repository. It allows you to configure basic security and other settings for everyone in your repository quickly. The Everyone group can be viewed in Repository Administration. It cannot be deleted or renamed, and its membership cannot be changed, but it otherwise behaves like any other group in Laserfiche.
