Laserfiche API Preview

Welcome to the Laserfiche API Preview! This preview release includes an alpha set of RESTful web APIs for Laserfiche Solution Providers and Technology Alliance Partners to build custom integrations between the Laserfiche Cloud repository and other third-party applications. This page will serve as your central hub for all of the Laserfiche API Preview resources including: how-to-guides, API reference material, interactive Swagger documentation and a snapshot of our API roadmap. We will continuously release more API functionality throughout the API Preview period, so follow along on our roadmap and check back to see when new APIs go live. We’d love to gather your feedback and build a robust web API to help support your integration needs in Laserfiche Cloud.
