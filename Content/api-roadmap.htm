Laserfiche API Feature Roadmap

This page provides a preview on work we have planned for the Laserfiche API. It includes a number of items, but is not a comprehensive feature list, and is intended to provide a look at what's coming so that you can provide feedback on which features you'd like to see. Please fill out this quick survey to tell us what you think.

Items are sorted by current status and include the following icons:

  • No icon - Planned
  • - In Progress
  • - Completed

Here's what we have planned throughout the preview:

  • Write tag data (2021.1)
  • Write link data (2021.1)
  • Delete entry (2021.1)
  • Copy entry (2021.1)
  • Retrieve simple search listing (2021.1)
  • Retrieve full search listing (2021.1)
  • Service Accounts - a new license type designed specifically for integrations to interact with the repository in the background

Here’s where we’d like to go after our commercial release:

  • OAuth 2.0 support
  • Export images as PDF and burn annotations
  • Export image thumbnail
  • Retrieve record properties
  • Set record properties
  • Streamed import
  • Entry locking
  • Apply annotations
  • Briefcase import
  • Retrieve record property definitions
  • Invoke Workflow instance
  • Read Lookup Table data
  • Write Lookup Table data
  • Delete Lookup Table data
  • Submit business process

Don't see what you're looking for on the roadmap? Please let us know by taking the survey.
