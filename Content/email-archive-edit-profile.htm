Editing an Profile

To edit an profile

  1. Open the Configuration dialog box from the Windows Start Menu.
  2. Select the profile you want to edit.
  3. Click the Edit profile button, or select Edit profile under Profile in the main menu.
  4. Configure profile options as desired.

To delete a profile

  1. Open the Configuration dialog box from the Windows Start Menu.
  2. Select the profile you want to edit.
  3. Click the Delete profile button, or select Delete profile under Profile in the main menu.
