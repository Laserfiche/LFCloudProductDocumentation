Verifying if You Have a Mailbox in Exchange Online

The Laserfiche Outlook Add-in for Microsoft 365 requires a user mailbox in Exchange Online. If you are using Office 365, but are connecting to Exchange Server (on-premises), you will not be able to access the Laserfiche Outlook Add-in for Microsoft 365.

The following steps show how to check your Outlook connection information.

  1. When Outlook is running, CTRL + right-click on the Outlook icon in the Windows taskbar notification area ().
  2. In the context menu, click Connection Status to open the Outlook Connection Status dialog box.
  3. Under the Server name column, verify that the appropriate mailbox is connecting to Office365.com. If the mailbox is connecting to a corporate domain, that indicates that you are connecting to Exchange Server (on-premises), and will not be able to use the Laserfiche Outlook Add-in for Microsoft 365.
