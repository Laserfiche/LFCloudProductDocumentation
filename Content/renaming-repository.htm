Renaming a Repository

Use the Plan page in Laserfiche Cloud Account Administration to rename a repository.

To rename a repository

  1. From the App Picker, select Account to view Laserfiche Cloud Account Administration. (The service is available for users with the Account Administration access right).
  2. In the navigation sidebar, select the Plan page.
  3. Under the Usage section, locate the repository. By default, the repository may be named Repository 1. Click on the Rename option to rename the display name of the repository.
