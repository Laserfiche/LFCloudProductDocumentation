March 6, 2024

Account Administration

  • As part of service stability optimizations, account administrators can no longer choose to bulk email multiple users at the same time. The changes include:
    • The Verify email and Send password email options have been removed from the toolbar and right-click shortcut menu. Administrators can click on the "Verify" link in the email column to send a verification email to a user and users can verify their email from their profile page. Users can reset their password using the Forgot password option on the sign-in page.
    • The Import from CSV feature's Import Users dialog no longer includes the Send a password setup email and Send email verification options. Administrators can choose to include a password for users in the CSV file. Newly imported users can reset their password using the Forgot password option on the sign-in page. If so desired, administrators can also select the Force password change option on the new users after the import process.
    • When deleting an identity provider, the action will not send an email to affected users. Affected users can reset their password using the Forgot password option on the sign-in page.
    • The Force password change option now resets passwords without sending an email. Affected users can reset their password during the next sign-in attempt.
