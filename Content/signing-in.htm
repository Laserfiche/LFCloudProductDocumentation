Signing In to Laserfiche

Laserfiche Cloud is an online content management system comprised of multiple web services. The following tables list out common web addresses for accessing Laserfiche Cloud services with a web browser.

Web Address Description

https://signin.laserfiche.com

https://signin.laserfiche.ca

https://signin.eu.laserfiche.com

 Default sign-in address.
https://signin.laserfiche.com/?customerid=111111111

https://signin.laserfiche.ca/?customerid=1111111111

https://signin.eu.laserfiche.com/?customerid=2111111111

 Sign-in URL to a specific account ID. Replace the italicized value with your Laserfiche Cloud account ID (9 or 10 digits). The sign-in page will have your account ID already populated.

https://app.laserfiche.com/laserfiche/?repo=r-000ab000

https://app.laserfiche.ca/laserfiche/?repo=r-000ab000

https://app.eu.laserfiche.com/laserfiche/?repo=r-000ab000

If you want to directly sign in and view your Laserfiche documents. Replace 000ab000 with your repository ID. This is the address that can be used in other components, such as Laserfiche Connector, when creating a profile that connects to the Laserfiche web client in Laserfiche Cloud.

When setting up a Laserfiche Connector profile that uses the Web Client URL and Repository option to connect to Laserfiche Cloud, set web client URL to https://app.laserfiche.com/laserfiche and the repository to r-000ab000, replacing 000ab000 with your repository ID.

https://r-000ab000.laserfiche.com

https://r-000ab000.laserfiche.ca

https://r-000ab000.eu.laserfiche.com

Use when prompted for server and repository information in Quick Fields. Replace 000ab000 with your repository ID.

Use when creating a Session object in Laserfiche SDK (Repository Access) Integrations. Replace 000ab000 with your repository ID.

https://portal.laserfiche.com/Portal/Welcome.aspx?repo=r-000ab000

https://portal.laserfiche.ca/Portal/Welcome.aspx?repo=r-000ab000

https://portal.eu.laserfiche.com/Portal/Welcome.aspx?repo=r-000ab000

The public portal for the specified account. Replace 000ab000 with your repository ID.

Note: The repository ID can be found on the Plan page in Account Administration.

To sign in to Laserfiche Cloud, please have your Laserfiche Cloud account ID (9 or 10 digits) ready as well as a valid user name and password.

If you are an administrator and this is your first time signing in to Laserfiche Cloud, please see Getting Started with Laserfiche Account Administration.

Recommended Web Browsers

For an optimized browsing experience, Laserfiche Cloud is best viewed with the latest versions of the Chrome and Firefox browsers for PC over a broadband Internet connection.
