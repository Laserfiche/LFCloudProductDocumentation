Configuring Single Sign-On with TD Ameritrade Veo One

Laserfiche Cloud supports linking Laserfiche Cloud user accounts to TD Ameritrade Veo One user accounts to facilitate viewing Laserfiche from within the Veo One platform. The configuration process involves the following steps.

  1. Contact TD Ameritrade to enable the Laserfiche Veo One integration.
  2. In Laserfiche Cloud Account Administration:
    1. Create Laserfiche Cloud user accounts that will be linked with Veo One user accounts.
    2. Configure single sign-on settings and map Laserfiche Cloud user account names to Veo One user account names.

Configuring Laserfiche Cloud for Single Sign-On

Sign in to Laserfiche Cloud with an administrator account.

  1. On the Settings page, click the Single Sign-On tab.
  2. Select Veo One from the Type drop-down list and save.
  3. Under the User Mapping section, click Add to see a list of Laserfiche Cloud user accounts. In the Add User Mappings dialog box, select the users that you want to link to Veo One user IDs and click OK.
  4. For each user, in the Veo One ID column, type the appropriate Veo One username.

  5. Click Save Changes when finished.

Note: Linked user accounts cannot sign in though the Laserfiche Cloud sign-in page. Users can sign in to the TD Ameritrade Veo One platform to view Laserfiche documents.
