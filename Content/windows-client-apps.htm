Connecting to Laserfiche Cloud From Laserfiche Windows Client and Laserfiche Quick Fields

To sign in to Laserfiche Cloud, please have your Laserfiche Cloud 9-digit account ID ready as well as a valid user name and password.

Laserfiche Windows Client

  1. Open the Laserfiche Client. If you do not see a location to type in your Laserfiche Cloud Account ID, click Sign in here next to Are you using Laserfiche Cloud?
  2. On the Sign-In page, enter the following
    • Account ID: Enter your Laserfiche Cloud 9-digit account ID (provided by your administrator).
    • User name: Enter your user name.
    • Password: Enter your password.
  3. Click Sign in.

Laserfiche Quick Fields

When prompted for server and repository information:

  1. Next to both Server and Repository, type r-repositoryid.laserfiche.com. The two values must match. You can find the repositoryid value in the Laserfiche Cloud Account Administration site under the Plan tab (if you are signed in as an administrator) or in Repository Administration under Summary (if signed in as a user with appropriate access).
  2. Select Password authentication and enter your user name and password.

    Note: The Windows authentication option cannot be used to connect to Laserfiche Cloud.

  3. Select Use SSL connection.

Laserfiche SDK (Repository Access) Integrations

When creating the Session object:

  1. When using the Session.Create overload that takes a CloudTicket object, for the repositoryHost string parameter, pass in the repository ID host name: r-123456789.laserfiche.com. Replace 123456789 with your repository ID.

    2. Note: The repository ID can be seen on the Plan page of the Laserfiche Cloud Administration site, the Overview page of the Repository Administration site, and in your web browser address bar when viewing your Laserfiche documents.
